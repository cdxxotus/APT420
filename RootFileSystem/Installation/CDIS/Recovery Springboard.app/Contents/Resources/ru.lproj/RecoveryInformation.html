Восстановление macOS на Mac с чипом Apple

Восстановление macOS — это встроенная система, которая переводит компьютер Mac в режим восстановления.

С помощью приложений режима восстановления macOS на компьютере Mac с процессором Apple можно исправить внутреннее хранилище, переустановить macOS, восстановить файлы из резервной копии Time Machine, настроить политику безопасности для разных томов, перенести файлы между двумя компьютерами Mac, выполнить загрузку в безопасном режиме и выполнить другие действия.

Для переустановки macOS требуется подключение к интернету. Можно использовать как проводное подключение к интернету, так и беспроводное. При попытке подключения к порталу, требующему авторизации (например, в кофейне, библиотеке или отеле), или к корпоративной сети, возможно, не удастся получить доступ к интернету в режиме восстановления macOS. См. раздел Подключение к интернету по Wi-Fi.

Примечание. Если Вы хотите подготовить компьютер Mac к продаже, обмену на скидку или отдать свой компьютер кому-то другому, воспользуйтесь приложением «Ассистент стирания» в macOS. Чтобы использовать приложение «Ассистент стирания», на компьютере Mac загрузите операционную систему macOS, выберите меню Apple  > «Системные настройки», в боковом меню нажмите «Основные»  (возможно, потребуется прокрутить вниз), справа нажмите «Перенос или сброс», нажмите кнопку «Стереть контент и настройки», затем следуйте инструкциям на экране.

Приложения, доступные в режиме восстановления macOS на компьютере Mac с чипом Apple

В режиме восстановления macOS на компьютере Mac с чипом Apple доступны следующие приложения.

  • Восстановление. Приложение «Восстановление» предоставляет доступ к другим приложениям в режиме восстановления macOS. См. разделы Загрузка компьютера в режиме восстановления macOS, Загрузка компьютера в режиме восстановления macOS и подключение устройств ввода через Bluetooth и Завершение сеанса в режиме восстановления macOS.

  • Восстановление системы из Time Machine. Вы можете восстановить данные из резервной копии Time Machine. В приложении «Восстановление» выберите «Восстановление из Time Machine», затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Восстановление системы из Time Machine» > «Завершить Восстановление системы из Time Machine». См. раздел Восстановление файлов с помощью Time Machine.

  • Установка macOS. Вы можете переустановить macOS на своем компьютере. В приложении «Восстановление» выберите «Переустановить macOS [название]», затем нажмите «Продолжить». Чтобы вернуться в приложение «Восстановление», выберите «Установка macOS [название]» > «Завершить Установку macOS». См. раздел Переустановка macOS.

  • Safari. В приложении «Восстановление» выберите Safari, затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Safari» > «Завершить Safari». При наличии подключения к интернету можно использовать Safari для поиска дополнительной информации на сайте службы поддержки Apple.

  • Дисковая утилита. Вы можете исправить или стереть внутреннее хранилище. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Дисковая утилита» > «Завершить Дисковую утилиту». См. разделы Стирание и переустановка macOS и Исправление внутреннего хранинилища.

  • Утилита безопасной загрузки. Вы можете настроить политики безопасности для компьютера Mac. В приложении «Восстановление» выберите «Утилиты» > «Утилита безопасной загрузки». Чтобы вернуться в приложение «Восстановление», выберите «Загрузочный диск» > «Завершить Загрузочный диск». См. раздел Изменение политики безопасности.

  • Терминал. Вы можете изменять настройки, используя командную строку. В приложении «Восстановление» выберите «Утилиты» > «Терминал». Для возврата в приложение «Восстановление» выберите «Терминал» > «Завершить Терминал».

  • Общий диск. Вы можете открыть общий доступ к диску на компьютере Mac, который загружен в режиме восстановления macOS. В приложении «Восстановление» выберите «Утилиты» > «Общий диск». См. раздел Перенос файлов между двумя компьютерами Mac с помощью приложения «Общий диск».

  • Загрузочный диск. Вы можете настроить загрузочный диск для компьютера Mac. Выберите меню Apple  > «Загрузочный диск». Чтобы завершить приложение, выберите «Загрузочный диск» > «Завершить Загрузочный диск».

Режим восстановления macOS можно также использовать для следующих задач:

  • Загрузка компьютера в безопасном режиме

  • Настройка загрузочного тома по умолчанию и перезагрузка компьютера

  • Просмотр журнала восстановления

  • Создание файла диагностики восстановления

Загрузка компьютера в режиме восстановления macOS

  1. На Mac выберите меню Apple  > «Выключить».

    Дождитесь полного выключения Mac. Компьютер Mac полностью выключен тогда, когда погасли экран и все индикаторы (в том числе панель Touch Bar и подсветка клавиатуры).

  2. Нажмите кнопку питания на Mac и удерживайте ее, пока не отобразятся системный том и кнопка «Параметры».

  3. Нажмите кнопку «Параметры», затем нажмите «Продолжить».

  4. Если отобразится запрос, выберите том для восстановления, затем нажмите «Далее».

  5. Выберите учетную запись с правами администратора, затем нажмите «Далее».

  6. Введите пароль учетной записи с правами администратора, затем нажмите «Продолжить».

    Когда приложение «Восстановление» отображается в строке меню, в окне или строке меню можно выбрать любой доступный параметр.

Загрузка компьютера в режиме восстановления macOS и подключение устройств ввода через Bluetooth

Следуйте этим инструкциям, чтобы объединить компьютер Mac в пару с Bluetooth-клавиатурой, трекпадом или мышью при загрузке в режиме восстановления macOS.

  1. На Mac выберите меню Apple  > «Выключить».

    Дождитесь полного выключения Mac. Компьютер Mac полностью выключен тогда, когда погасли экран и все индикаторы (в том числе панель Touch Bar и подсветка клавиатуры).

  2. Нажмите кнопку питания на Mac и удерживайте ее, пока не отобразятся системный том и кнопка «Параметры».

  3. Нажмите кнопку питания три раза.

    Примечание. Нажимать нужно с промежутками в 1 секунду. Однако не следует нажимать кнопку питания слишком быстро, так как быстрые нажатия предназначены для вызова других функций, таких как VoiceOver.

  4. Используйте приложение «Ассистент настройки Bluetooth», чтобы объединить аксессуары Bluetooth в пару с компьютером Mac.

    Примечание. Приложение «Ассистент настройки Bluetooth» не отображается, если устройство ввода уже подключено.

  5. Выберите учетную запись с правами администратора, затем нажмите «Далее».

  6. Введите пароль учетной записи с правами администратора, затем нажмите «Продолжить».

    Когда приложение «Восстановление» отображается в строке меню, в окне или строке меню можно выбрать любой доступный параметр.

Загрузка компьютера в безопасном режиме

  1. На Mac выберите меню Apple  > «Выключить».

    Дождитесь полного выключения Mac. Компьютер Mac полностью выключен тогда, когда погасли экран и все индикаторы (в том числе панель Touch Bar и подсветка клавиатуры).

  2. Нажмите кнопку питания на Mac и удерживайте ее, пока не отобразятся системный том и кнопка «Параметры».

  3. Выберите том.

  4. Нажмите и удерживайте клавишу Shift, затем нажмите «Продолжить в безопасном режиме».

    Компьютер перезагрузится автоматически.

Настройка загрузочного тома по умолчанию и перезагрузка компьютера

  1. На Mac выберите меню Apple  > «Выключить».

    Дождитесь полного выключения Mac. Компьютер Mac полностью выключен тогда, когда погасли экран и все индикаторы (в том числе панель Touch Bar и подсветка клавиатуры).

  2. Нажмите кнопку питания на Mac и удерживайте ее, пока не отобразятся системный том и кнопка «Параметры».

  3. Выберите том.

  4. Нажмите и удерживайте клавишу Option, затем нажмите «Всегда использовать».

    Компьютер перезагрузится автоматически.

Подключение к интернету по Wi-Fi

Если на значке меню Wi-Fi (в правом верхнем углу экрана) есть одна или несколько полос , функция Wi-Fi включена, а компьютер подключен к беспроводной сети. Если на значке меню Wi-Fi нет полос , функция Wi-Fi включена, но компьютер не подключен к беспроводной сети. Если значок меню Wi-Fi перечеркнут , функция Wi-Fi выключена. Перед подключением к беспроводной сети необходимо включить Wi-Fi.

На Mac выполните любое из следующих действий.

  • Включение Wi-Fi. Нажмите на значок меню Wi-Fi, затем выберите «Включить Wi-Fi».

  • Выключение Wi-Fi. Нажмите на значок меню Wi-Fi, затем выберите «Выключить Wi-Fi».

  • Подключение к сети. Нажмите на значок меню Wi-Fi, затем выберите сеть. При необходимости введите пароль сети.

  • Подключение к закрытой сети. Нажмите на значок меню Wi-Fi, затем выберите «Подключиться к другой сети». Введите имя сети и, если необходимо, ее пароль.

Исправление внутреннего хранилища

Если возникли проблемы с компьютером или если после загрузки компьютера отобразилось окно «Восстановление macOS», это может означать, что необходимо исправить ошибки в работе внутреннего хранилища.

  1. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить».

  2. В приложении «Дисковая утилита» выберите «Вид» > «Показывать все устройства».

  3. В боковой панели выберите том, затем нажмите кнопку «Первая помощь» .

  4. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  5. Когда процесс «Первая помощь» завершится, нажмите «Готово».

  6. Повторите шаги с 3-го по 5-й для каждого тома на устройстве хранения.

  7. В боковой панели выберите контейнер, затем нажмите кнопку «Первая помощь» .

  8. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  9. Когда процесс «Первая помощь» завершится, нажмите «Готово».

  10. Повторите шаги с 7-го по 9-й для каждого контейнера на устройстве хранения.

  11. В боковой панели выберите устройство хранения, затем нажмите кнопку «Первая помощь» .

  12. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  13. Когда процесс «Первая помощь» завершится, нажмите «Готово».

Если диск не удается исправить с помощью Дисковой утилиты, по возможности создайте резервные копии всей своей информации, затем следуйте инструкциям в разделе Стирание и переустановка macOS.

Для возврата в приложение «Восстановление» выберите «Дисковая утилита» > «Завершить Дисковую утилиту».

Восстановление файлов с помощью Time Machine

Если ранее Вы создали резервную копию Time Machine, Вы можете восстановить файлы из резервной копии.

Важно! Используйте резервную копию Time Machine только для восстановления информации на компьютере, с которого была создана эта резервная копия. Если Вы хотите перенести информацию с этого Mac на новый компьютер, завершите сеанс в режиме восстановления macOS, перезагрузите Mac и используйте приложение «Ассистент миграции» для macOS.

  1. Если резервная копия находится на станции Time Capsule, убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» выберите «Восстановление из Time Machine», затем нажмите «Продолжить».

  3. Следуйте инструкциям на экране в приложении «Восстановление системы из Time Machine».

Переустановка macOS

В некоторых случаях может потребоваться переустановить macOS. При этом можно сохранить файлы и пользовательские настройки.

  1. Убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» выберите «Переустановить macOS [название]», затем нажмите «Продолжить».

  3. Следуйте инструкциям на экране.

    Важно! При запросе выбора диска выберите тот диск, который сейчас используется для запуска macOS (обычно это единственный доступный диск).

Стирание и переустановка macOS

Стирание и переустановка macOS приводят к удалению учетных записей пользователя, сетевых настроек, а также всех файлов и папок на Mac. Прежде чем приступать к стиранию macOS, попытайтесь снова войти в систему, затем создайте резервные копии всех файлов, которые Вы хотите сохранить, скопировав их на устройство хранения. Кроме того, запишите данные сетевых настроек из раздела настроек «Сеть», чтобы было проще подключиться к интернету и другим сетям после переустановки macOS.

Примечание. После стирания и переустановки macOS можно установить более раннюю версию macOS. Чтобы обновить текущую установленную версию до новейшей версии macOS, выберите меню Apple  > «Системные настройки», в боковом меню нажмите «Основные»  (возможно, для этого придется прокрутить вниз), затем справа нажмите «Обновление ПО».

  1. Убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить».

  3. В боковой панели приложения «Дисковая утилита» выберите том, который нужно стереть, затем нажмите кнопку «Стереть» в панели инструментов.

  4. Введите имя тома в поле «Имя».

  5. Во всплывающем меню «Формат» выберите APFS, затем нажмите «Стереть группу томов».

  6. Когда стирание завершится, нажмите «Готово», затем выберите «Дисковая утилита» > «Завершить Дисковую утилиту».

  7. В приложении «Восстановление» выберите «Переустановить macOS [название]», нажмите «Продолжить», затем следуйте инструкциям на экране.

Изменение политики безопасности

  1. В приложении «Восстановление» выберите «Утилиты» > «Утилита безопасной загрузки».

  2. Выберите систему, которую хотите использовать для настройки политики безопасности.

  3. Если диск защищен шифрованием, нажмите «Снять защиту», введите пароль, затем нажмите «Снять защиту».

  4. Нажмите «Политика безопасности».

  5. Выберите один из следующих вариантов.

    • Высший уровень безопасности. Можно запускать только текущую операционную систему или подписанное ПО операционной системы, которое на данный момент признано Apple надежным. Если выбрать этот вариант, для установки ПО потребуется подключение к интернету.

    • Сниженный уровень безопасности. Можно запускать любую версию подписанного ПО операционной системы, когда-либо признанную Apple надежной.

  6. Если Вы выбрали вариант «Сниженный уровень безопасности», при необходимости выберите одну из настроек ниже.

    • Разрешить пользователям управлять расширениями ядра от подтвержденных разработчиков. Эта настройка разрешает установку программного обеспечения, которое использует устаревшие расширения ядра.

    • Разрешить удаленное управление расширениями ядра и автоматическую установку обновлений ПО. Эта настройка разрешает удаленное управление устаревшими расширениями ядра и обновлениями ПО с помощью решения для управления мобильными устройствами (MDM).

  7. Нажмите «ОК».

  8. Если Вы внесли изменения в политику безопасности, нажмите всплывающее меню «Пользователь», выберите учетную запись администратора, введите пароль учетной записи администратора, затем нажмите «ОК».

  9. Выберите меню Apple  > «Перезагрузить».

    Чтобы изменения вступили в силу, необходимо перезагрузить Mac.

Перенос файлов между двумя компьютерами Mac с помощью приложения «Общий диск»

Можно переносить файлы между Mac, загруженным в режиме восстановления, и другим Mac. Для этого необходимо открыть общий доступ к тому и подключить компьютеры друг к другу, используя кабель USB, USB-C или Thunderbolt.

  1. Подключите компьютеры друг к другу, используя кабель USB, USB-C или Thunderbolt.

  2. В приложении «Восстановление» выберите «Утилиты» > «Общий диск».

  3. Выберите том, к которому хотите открыть общий доступ, затем нажмите «Открыть общий доступ».

  4. На другом Mac откройте окно Finder, затем в разделе «Места» бокового меню нажмите «Сеть».

  5. В окне «Сеть» дважды нажмите на Mac, на котором находится общий диск, нажмите «Подключиться как», выберите «Как гость» в окне «Подключение», затем нажмите «Подключиться».

  6. Перенесите файлы.

  7. Когда перенос файлов будет завершен, извлеките том на другом Mac.

  8. В приложении «Общий диск» нажмите «Закрыть общий доступ».

  9. Выберите «Общий диск» > «Завершить Общий диск».

Просмотр журнала восстановления

Журнал восстановления содержит сообщения, записанные в режиме восстановления macOS. Вы можете указать типы сообщений, которые хотите отобразить, а также фильтровать сообщения, чтобы найти определенные слова, или сохранить журнал восстановления в виде файла.

  1. В приложении «Восстановление» выберите «Окно» > «Журнал Восстановления».

  2. Во всплывающем меню «Уровень детализации» выберите тип отображаемых сообщений.

    • Показать только ошибки. Выберите этот параметр, чтобы отобразились только сообщения об ошибках.

    • Показать ошибки и выполнение. Выберите этот параметр, чтобы отобразились сообщения об ошибках и о процессе выполнения.

    • Показать все сообщения. Выберите этот параметр, чтобы отобразились все сообщения.

  3. Чтобы отфильтровать сообщения, введите в поле «Фильтр» текст, который нужно найти.

  4. Чтобы сохранить журнал восстановления, нажмите кнопку «Сохранить», выберите место сохранения файла, затем нажмите «Сохранить».

  5. Когда закончите просматривать журнал восстановления, нажмите кнопку «Закрыть»  в левом верхнем углу окна.

Создание файла диагностики восстановления

Если Вы отправляли отчет об ошибке в приложении «Ассистент обратной связи», Вас могут попросить предоставить файл диагностики восстановления. Вы можете создать файл и сохранить его на внешнее устройство хранения.

  1. Подключите внешнее устройство хранения к своему компьютеру Mac.

  2. Нажмите Control-Option-Shift-Command-точка (.).

  3. Экран на мгновение погаснет. Подождите пару минут, пока не отобразится диалоговое окно.

  4. Используя диалоговое окно, можно сохранить файл диагностики восстановления на внешнем устройстве хранения.

Завершение сеанса в режиме восстановления macOS

Компьютер Mac перезагружается автоматически при восстановлении системы из Time Machine или переустановке macOS. Однако в некоторых случаях может понадобиться вручную завершить сеанс в режиме восстановления macOS (например, если Вы хотите сделать резервные копии файлов перед переустановкой macOS или если Вы только что изменили настройку через Утилиту безопасной загрузки).

На компьютере Mac выполните одно из следующих действий.

  • Перезагрузка Mac. Выберите меню Apple  > «Перезагрузить».

  • Выбор другого загрузочного диска перед перезагрузкой Mac. Выберите меню Apple  > «Загрузочный диск», выберите загрузочный диск, затем нажмите «Перезагрузить».

  • Выключение Mac. Выберите меню Apple  > «Выключить».
