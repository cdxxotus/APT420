Utiliser la fonctionnalité de récupération de macOS sur un Mac doté d’un processeur Intel

La fonctionnalité de récupération de macOS est le système de récupération intégré de votre Mac.

Vous pouvez utiliser les apps disponibles dans la fonctionnalité de récupération de macOS pour réparer votre périphérique de stockage interne, réinstaller macOS, restaurer vos fichiers à partir d’une sauvegarde Time Machine, définir les options de sécurité, et plus encore.

La réinstallation de macOS nécessite une connexion à Internet. Pour vous connecter à Internet, vous pouvez utiliser une connexion réseau sans fil ou câblée. Si vous essayez de vous connecter à un réseau Wi-Fi via un portail captif (dans un café, une bibliothèque ou un hôtel, par exemple) ou à un réseau d’entreprise, il se peut que vous ne puissiez pas accéder à Internet dans la fonctionnalité de récupération de macOS. Consultez la rubrique Se connecter à Internet en Wi-Fi.

Remarque : si vous souhaitez préparer votre Mac dans l’optique de l’échanger, de le vendre ou de le donner, et s’il dispose d’un processeur Intel avec une puce Apple T2 Security, utilisez Assistant d’effacement sous macOS. Pour ce faire, démarrez votre ordinateur avec macOS, choisissez le menu Pomme  > Réglages Système, cliquez sur Général  dans la barre latérale (vous devrez peut-être faire défiler vers le bas), cliquez sur « Transférer ou réinitialiser » à droite, cliquez sur le bouton « Effacer contenu et réglages », puis suivez les instructions à l’écran.

Apps disponibles dans la fonctionnalité de récupération de macOS sur un Mac doté d’un processeur Intel

Les apps suivantes sont disponibles dans la fonctionnalité de récupération de macOS sur un Mac doté d’un processeur Intel :

  • Récupération : l’app Récupération s’ouvre lorsque votre ordinateur démarre à partir de la fonctionnalité de récupération de macOS. Consultez les rubriques Démarrer un Mac doté d’un processeur Intel à partir de la fonctionnalité de récupération de macOS et Quitter la fonctionnalité de récupération de macOS.

  • Restauration système Time Machine : restaurez vos données à partir d’une sauvegarde Time Machine. Dans l’app Récupération, sélectionnez « Restaurer à partir de Time Machine », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Restauration système Time Machine > Quitter la restauration système Time Machine. Consultez la rubrique Utiliser Time Machine pour restaurer vos fichiers.

  • Installer macOS : réinstallez macOS sur votre ordinateur. Dans l’app Récupération, sélectionnez « Réinstaller macOS [nom] », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Installer macOS [nom] > Quitter Installer macOS. Consultez la rubrique Réinstaller macOS.

  • Safari : dans l’app Récupération, sélectionnez Safari, puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Safari > Quitter Safari. Si vous êtes connecté à Internet, vous pouvez utiliser Safari pour rechercher de plus amples informations sur le site web d’assistance Apple.

  • Utilitaire de disque : réparez ou effacez votre disque. Dans l’app Récupération, sélectionnez « Utilitaire de disque », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Utilitaire de disque > Quitter Utilitaire de disque. Consultez la rubrique Effacer et réinstaller macOS.

  • Utilitaire Sécurité au démarrage : définissez les règlements de sécurité de votre Mac. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage. Pour revenir à l’app Récupération, choisissez Utilitaire Sécurité au démarrage > Quitter Utilitaire Sécurité au démarrage. Consultez les rubriques Utiliser Utilitaire Sécurité au démarrage pour définir un mot de passe pour le programme interne et Utiliser Utilitaire Sécurité au démarrage pour définir les options de démarrage sécurisé.

    Les options disponibles dans Utilitaire Sécurité au démarrage varient en fonction du modèle de votre Mac. Consultez les articles À propos de l’utilitaire Sécurité au démarrage sur un Mac doté de la puce Apple T2 Security et Ordinateurs Mac dotés de la puce Apple T2 Security de l’assistance Apple.

  • Disque de démarrage : définissez le disque de démarrage de votre Mac. Choisissez le menu Pomme  > Disque de démarrage. Pour revenir à l’app Récupération, choisissez Disque de démarrage > Quitter Disque de démarrage.

  • Terminal : dans l’app Récupération, choisissez Utilitaires > Terminal. Pour revenir à l’app Récupération, choisissez Terminal > Quitter Terminal.

Vous pouvez également utiliser la fonctionnalité de récupération de macOS pour effectuer les tâches suivantes :

  • Consulter l’historique de récupération

  • Créer un fichier de diagnostic de récupération

Démarrer un Mac doté d’un processeur Intel à partir de la fonctionnalité de récupération de macOS

  1. Redémarrez votre Mac.

  2. Appuyez immédiatement sur l’une des combinaisons de touches suivantes et maintenez les touches enfoncées jusqu’à ce que l’écran de démarrage apparaisse :

    • Commande + R : démarrez à partir du système de récupération de macOS intégré à votre Mac. Utilisez cette combinaison de touches pour réinstaller la dernière version de macOS installée sur votre système, ou pour utiliser les autres apps disponibles dans la fonctionnalité de récupération de macOS.

    • Option + Commande + R : démarrez à partir de la fonctionnalité de récupération de macOS sur Internet. Utilisez cette combinaison de touches pour réinstaller macOS et effectuer la mise à niveau vers la dernière version de macOS compatible avec votre Mac.

    • Option + Maj + Commande + R : démarrez à partir de la fonctionnalité de récupération de macOS sur Internet. Utilisez cette combinaison de touches pour réinstaller la version de macOS fournie avec votre Mac ou la version la plus proche encore disponible.

  3. Si un cadenas est affiché, saisissez le mot de passe de votre Mac.

  4. Si plusieurs volumes sont présents sur votre disque, sélectionnez celui que vous souhaitez récupérer, puis cliquez sur Suivant.

  5. Si vous y êtes invité, choisissez un compte administrateur, cliquez sur Suivant, saisissez le mot de passe de ce compte, puis cliquez sur Continuer.

    Lorsque l’app Récupération apparaît dans la barre des menus, vous pouvez choisir l’une des options disponibles dans la fenêtre ou dans la barre des menus.

Se connecter à Internet en Wi-Fi

Si l’icône de menu Wi-Fi (dans le coin supérieur droit de l’écran) contient une ou plusieurs barres , cela signifie que le Wi-Fi est activé et connecté à un réseau sans fil. Si l’icône de menu Wi-Fi ne contient aucune barre , cela signifie que le Wi-Fi est activé mais n’est pas connecté à un réseau sans fil. Si l’icône de menu Wi-Fi est barrée , cela signifie que le Wi-Fi est désactivé. Le Wi-Fi doit être activé avant que vous puissiez vous connecter à un réseau sans fil.

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Pour activer le Wi-Fi : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Activer le Wi-Fi ».

  • Pour désactiver le Wi-Fi : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Désactiver le Wi-Fi ».

  • Pour vous connecter à un réseau : cliquez sur l’icône de menu Wi-Fi, puis choisissez un réseau. Si vous y êtes invité, saisissez le mot de passe du réseau.

  • Pour vous connecter à un réseau fermé : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Se connecter à un autre réseau ». Saisissez le nom du réseau et, si vous y êtes invité, le mot de passe pour y accéder.

Réparer le périphérique de stockage interne

Si vous rencontrez des problèmes avec votre ordinateur ou si la fenêtre de la fonctionnalité de récupération de macOS s’est affichée lors de son démarrage, il se peut que vous deviez réparer son périphérique de stockage interne (par exemple, un SSD).

  1. Dans l’app Récupération, sélectionnez « Utilitaire de disque », puis cliquez sur Continuer.

  2. Dans l’app Utilitaire de disque, choisissez Présentation > Afficher tous les appareils.

  3. Dans la barre latérale, sélectionnez un volume, puis cliquez sur le bouton S.O.S. .

  4. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  5. Une fois le processus S.O.S. terminé, cliquez sur OK.

  6. Recommencez les étapes 3 à 5 pour chaque volume du périphérique de stockage.

  7. Dans la barre latérale, sélectionnez un conteneur, puis cliquez sur le bouton S.O.S. .

  8. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  9. Une fois le processus S.O.S. terminé, cliquez sur OK.

  10. Recommencez les étapes 7 à 9 pour chaque conteneur du périphérique de stockage.

  11. Dans la barre latérale, sélectionnez le périphérique de stockage, puis cliquez sur le bouton S.O.S. .

  12. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  13. Une fois le processus S.O.S. terminé, cliquez sur OK.

Si Utilitaire de disque n’est pas en mesure de réparer votre disque, sauvegardez autant d’informations que possible, puis suivez les instructions de la rubrique Effacer et réinstaller macOS.

Pour revenir à l’app Récupération, choisissez Utilitaire de disque > Quitter Utilitaire de disque.

Utiliser Time Machine pour restaurer vos fichiers

Si vous avez déjà créé une sauvegarde Time Machine, vous pouvez restaurer vos fichiers depuis la sauvegarde.

Important : n’utilisez votre sauvegarde Time Machine que pour restaurer des données sur l’ordinateur qui est la source de cette sauvegarde. Si vous souhaitez transférer des données sur un nouvel ordinateur, quittez la fonctionnalité de récupération de macOS, redémarrez votre Mac, puis utilisez l’app Assistant migration dans macOS.

  1. Si votre sauvegarde se trouve sur une Time Capsule, vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, sélectionnez « Restaurer à partir de Time Machine », puis cliquez sur Continuer.

  3. Dans l’app Restauration système Time Machine, suivez les instructions à l’écran.

Utiliser Utilitaire Sécurité au démarrage pour définir un mot de passe pour le programme interne

La définition d’un mot de passe pour le programme interne vous permet d’empêcher votre Mac de démarrer à partir d’un périphérique de stockage interne ou externe autre que le disque de démarrage sélectionné si vous ne saisissez pas le mot de passe du programme interne. Vous pouvez activer, désactiver ou modifier le mot de passe du programme interne à l’aide d’Utilitaire Sécurité au démarrage.

Lorsque le mot de passe du programme interne est activé, vous devez le saisir chaque fois que vous démarrez votre Mac à partir de la fonctionnalité de récupération de macOS ou d’un périphérique de stockage autre que celui sélectionné dans les réglages « Disque de démarrage ».

Les modifications apportées au mot de passe du programme interne sont appliquées au redémarrage de votre Mac.

  1. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage.

  2. Si vous y êtes invité, saisissez le nom d’utilisateur et le mot de passe d’un administrateur.

  3. Effectuez l’une des opérations suivantes :

    • Pour activer le mot de passe du programme interne : cliquez sur « Activer le mot de passe du programme interne », saisissez un nouveau mot de passe, saisissez-le à nouveau pour le confirmer, puis cliquez sur « Définir le mot de passe ».

    • Pour désactiver le mot de passe du programme interne : cliquez sur « Désactiver le mot de passe du programme interne », saisissez le mot de passe, puis cliquez sur « Désactiver le mot de passe ».

    • Pour modifier le mot de passe du programme interne : cliquez sur « Modifier le mot de passe », saisissez l’ancien mot de passe, saisissez un nouveau mot de passe et confirmez-le, puis cliquez sur « Modifier le mot de passe ».

  4. Cliquez sur « Quitter Utilitaire Sécurité au démarrage ».

Utiliser Utilitaire Sécurité au démarrage pour définir les options de démarrage sécurisé

Les options de démarrage sécurisé ne sont disponibles que sur les ordinateurs Mac dotés de la puce Apple T2 Security.

Les modifications apportées aux options de démarrage sécurisé sont appliquées au redémarrage de votre Mac.

  1. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage.

  2. Saisissez le nom d’utilisateur et le mot de passe d’un administrateur.

  3. Sélectionnez une option de sécurité sous « Démarrage sécurisé ».

    • Sécurité maximale : garantit que seul votre système d’exploitation actuel, ou le logiciel de système d’exploitation signé actuellement approuvé par Apple, peut s’exécuter. Ce mode requiert une connexion réseau au moment de l’installation des logiciels.

    • Sécurité normale : autorise l’exécution de toute version d’un logiciel de système d’exploitation signé ayant déjà été approuvé par Apple.

    • Aucune sécurité : n’applique aucune mesure de sécurité au système d’exploitation démarrable.

  4. Sélectionnez une option de démarrage sous « Support de démarrage autorisé ».

    • Ne pas autoriser le démarrage à partir de supports externes ou amovibles : empêche le démarrage depuis tout périphérique non protégé par la puce Apple T2 Security, comme les disques USB et Thunderbolt ou les disques PCIe ou SATA internes.

    • Autoriser le démarrage à partir de supports externes ou amovibles : autorise le démarrage depuis n’importe quel périphérique.

  5. Choisissez Utilitaire Sécurité au démarrage > Quitter Utilitaire Sécurité au démarrage.

Réinstaller macOS

Dans certains cas, il peut être nécessaire de réinstaller macOS. Cette réinstallation peut s’effectuer en gardant intacts vos fichiers et réglages utilisateur.

  1. Vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur « Réinstaller macOS [nom] », puis sur Continuer.

  3. Suivez les instructions à l’écran.

    Important : lorsqu’il vous est demandé de sélectionner un disque, sélectionnez votre disque macOS actuel (dans la plupart des cas, il s’agit du seul disque disponible).

Effacer et réinstaller macOS

Si vous effacez puis réinstallez macOS, tous vos comptes utilisateur, réglages réseau, fichiers et dossiers sont supprimés de votre Mac. Avant d’effacer macOS, essayez de vous reconnecter à votre système, puis sauvegardez tous les fichiers que vous souhaitez conserver en les copiant sur un autre périphérique de stockage. Notez également les réglages réseau définis dans les réglages Réseau pour vous reconnecter plus facilement après la réinstallation de macOS.

  1. Vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur « Utilitaire de disque », puis sur Continuer.

  3. Dans Utilitaire de disque, sélectionnez le volume que vous souhaitez effacer dans la barre latérale, puis cliquez sur le bouton Effacer dans la barre d’outils.

  4. Saisissez un nom pour le volume dans le champ Nom.

  5. Cliquez sur le menu local Format, choisissez APFS, puis cliquez sur « Effacer le groupe de volumes ».

  6. Une fois le processus d’effacement terminé, cliquez sur OK, puis choisissez Utilitaire de disque > Quitter Utilitaire de disque.

  7. Dans l’app Récupération, cliquez sur « Réinstaller macOS [nom] », cliquez sur Continuer, puis suivez les instructions à l’écran.

Consulter l’historique de récupération

L’historique de récupération contient les messages consignés par la fonctionnalité de récupération de macOS. Vous pouvez choisir les types de messages à afficher, filtrer les messages pour rechercher un texte précis, et enregistrer l’historique de récupération sous forme de fichier.

  1. Dans l’app Récupération, choisissez Fenêtre > Historique de Récupération.

  2. Cliquez sur le menu local « Niveau de détail », puis choisissez les messages à afficher :

    • Afficher uniquement les erreurs : choisissez cette option pour n’afficher que les messages d’erreur.

    • Afficher les erreurs et la progression : choisissez cette option pour afficher les messages d’erreur et de progression.

    • Afficher tout l’historique : choisissez cette option pour afficher tous les messages.

  3. Pour filtrer les messages, saisissez le texte à rechercher dans le champ Filtre.

  4. Pour enregistrer l’historique de récupération, cliquez sur le bouton Enregistrer, choisissez un emplacement pour le fichier, puis cliquez sur Enregistrer.

  5. Lorsque vous avez terminé de consulter l’historique de récupération, cliquez sur le bouton Fermer  dans le coin supérieur gauche de la fenêtre.

Créer un fichier de diagnostic de récupération

Si vous signalez un problème à l’aide d’Assistant d’évaluation, il peut vous être demandé de fournir un fichier de diagnostic de récupération. Vous pouvez créer ce fichier et l’enregistrer sur un périphérique de stockage externe.

  1. Connectez un périphérique de stockage externe à votre Mac.

  2. Appuyez sur Contrôle + Option + Maj + Commande + Point.

  3. Après le clignotement de l’écran, patientez quelques instants jusqu’à ce qu’une zone de dialogue apparaisse.

  4. Utilisez la zone de dialogue pour enregistrer le fichier de diagnostic de récupération dans un emplacement de votre périphérique de stockage externe.

Quitter la fonctionnalité de récupération de macOS

Votre Mac redémarre automatiquement lors de la restauration de votre système depuis Time Machine ou lors de la réinstallation de macOS. Dans les autres cas, vous devez quitter manuellement la fonctionnalité de récupération de macOS (par exemple, si vous souhaitez sauvegarder vos fichiers avant la réinstallation de macOS ou si vous avez seulement modifié un réglage à l’aide d’Utilitaire Sécurité au démarrage).

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Pour redémarrer votre Mac : choisissez le menu Pomme  > Redémarrer.

  • Pour choisir un autre disque de démarrage avant de redémarrer votre Mac : choisissez le menu Pomme  > Disque de démarrage, sélectionnez un disque de démarrage, puis cliquez sur Redémarrer.

  • Pour éteindre votre Mac : choisissez le menu Pomme  > Éteindre.
