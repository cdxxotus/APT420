Informations pour les utilisateurs invités sur Mac

Lorsque vous vous connectez en tant qu’invité à un Mac sur lequel l’option FileVault est activée, vous pouvez uniquement utiliser Safari. Vous ne pourrez utiliser aucune autre app sur le Mac.

Lorsque vous vous déconnectez, les fichiers que vous avez créés sont supprimés.

Se connecter à Internet

Si l’icône de menu Wi‑Fi (dans le coin supérieur droit de l’écran) contient une ou plusieurs barres , le Wi‑Fi est activé et connecté à un réseau sans fil. Si l’icône de menu Wi‑Fi ne contient aucune barre , le Wi‑Fi est activé, mais il n’est pas connecté à un réseau sans fil. Si une barre diagonale apparaît sur l’icône de menu Wi‑Fi , le Wi‑Fi est désactivé. Il faut activer celui-ci pour pouvoir accéder à un réseau sans fil.

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Activer ou désactiver le Wi-Fi : Cliquez sur l’icône de menu Wi‑Fi, puis choisissez Activer le Wi‑Fi ou Désactiver le Wi‑Fi.

  • Accéder à un réseau : Cliquez sur l’icône de menu Wi‑Fi, puis sélectionnez un réseau. Au besoin, saisissez le mot de passe du réseau.

Terminer la session d’un utilisateur invité

Lorsque vous avez fini d’utiliser le Mac, procédez de l’une des façons suivantes :

  • Redémarrer l’ordinateur : Choisissez le menu Pomme  > Redémarrer ou choisissez Safari > Quitter Safari.

  • Éteindre l’ordinateur : Choisissez le menu Pomme  > Éteindre.
