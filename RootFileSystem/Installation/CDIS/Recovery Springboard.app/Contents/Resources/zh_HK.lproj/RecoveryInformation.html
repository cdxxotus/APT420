在配備 Apple 晶片的 Mac 上使用 macOS 還原

「macOS 還原」是 Mac 的內置還原系統。

你可以在配備 Apple 晶片的 Mac 上使用「macOS 還原」中的 App 來修復電腦的內置儲存空間、重新安裝 macOS、從「時光機」備份還原檔案、為不同卷宗設定保安政策、在兩部 Mac 電腦之間傳送檔案，以及在安全模式開機等等。

重新安裝 macOS 需要互聯網連線。如要連接互聯網，你可以使用無線或有線網絡連線。如你正在嘗試透過限制入口網站（例如咖啡店、圖書館或酒店）或企業網絡連接 Wi-Fi，你可能無法在「macOS 還原」中取用互聯網。請參閲：使用 Wi-Fi 連接互聯網。

注意：如你想讓 Mac 準備好以舊換新、出售或贈送，請在 macOS 中使用「清除輔助程式」。如要使用「清除輔助程式」，請在 macOS 中啟動電腦，選擇「蘋果」選單 >「系統設定」，在側邊欄中按一下「一般」（你可能需要向下捲動），按一下右邊的「轉移或重設」，按一下「清除所有內容和設定」按鈕，然後按照螢幕上的指示操作。

配備 Apple 晶片的 Mac 上「macOS 還原」中可用的 App

以下的 App 可在配備 Apple 晶片的 Mac 上的「macOS 還原」中使用：

  • 還原：「還原」App 在「macOS 還原」中提供其他 App 的取用權。請參閲：在「macOS 還原」中開啟電腦、在「macOS 還原」中開啟電腦並配對藍牙輸入裝置和結束「macOS 還原」。

  • 時光機系統還原：從「時光機」備份還原資料。在「還原」App 中，選擇「從時光機還原」，然後按一下「繼續」。如要返回「還原」App， 選擇「時光機系統還原」>「結束時光機系統還原」。請參閲：使用「時光機」來還原你的檔案。

  • 安裝 macOS：在你的電腦上重新安裝 macOS。在「還原」App 中，選擇「重新安裝 macOS [名稱]」，然後按一下「繼續」。如要返回「還原」App， 選擇「安裝 macOS [名稱]」>「結束安裝 macOS」。請參閲：重新安裝 macOS。

  • Safari：在「還原」App 中，選擇 Safari，然後按一下「繼續」。如要返回「還原」App，選擇 Safari >「結束 Safari」。如你已連接到互聯網，你可以使用 Safari 來在 Apple 支援網站上搜尋更多資料。

  • 磁碟工具程式：修復或清除內置儲存空間。在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。請參閲：清除和重新安裝 macOS 和修復你的內置儲存空間。

  • 開機保安工具程式：為你的 Mac 設定保安政策。在「還原」App 中，選擇「工具程式」>「開機保安工具程式」。如要返回「還原」App， 選擇「開機磁碟」> 「結束開機磁碟」。請參閲：更改保安政策。

  • 終端機：透過命令列變更設定。在「還原」App 中，選擇「工具程式」>「終端機」。如要返回「還原」App， 選擇「終端機」> 「結束終端機」。

  • 分享磁碟：分享在「macOS 還原」中開機的 Mac 磁碟。在「還原」App 中，選擇「工具程式」>「分享磁碟」。請參閲：使用「分享磁碟」來在兩部 Mac 電腦之間傳送檔案。

  • 開機磁碟：為 Mac 設定開機磁碟。選擇「蘋果」選單 >「開機磁碟」。如要結束 App，選擇「開機磁碟」>「結束開機磁碟」。

你也可以使用「macOS 還原」來執行以下操作：

  • 在安全模式開機

  • 設定預設開機卷宗並重新啟動電腦

  • 檢視還原記錄

  • 製作還原診斷檔案

在「macOS 還原」中開機

  1. 在 Mac 上，選擇「蘋果」選單 >「關機」。

    等待 Mac 完全關閉。Mac 在黑色畫面和所有燈（包括觸控欄和鍵盤）關閉時即為完全關閉。

  2. 在 Mac 上按住電源按鈕，直到顯示系統卷宗和「選項」按鈕。

  3. 按一下「選項」按鈕，然後按一下「繼續」。

  4. 如被詢問，請選擇要還原的卷宗，然後按一下「下一步」。

  5. 選擇管理員帳户，然後按一下「下一步」。

  6. 輸入管理員帳户的密碼，然後按一下「繼續」。

    當「還原」App 在選單列上顯示，你可以選擇在視窗或選單列上任何可用的選項。

在「macOS 還原」中開啟電腦並配對藍牙輸入裝置

當你在「macOS 還原」中開機並需要配對 Bluetooth® 鍵盤、觸控板或滑鼠到 Mac 上，請遵循這些指引。

  1. 在 Mac 上，選擇「蘋果」選單 >「關機」。

    等待 Mac 完全關閉。Mac 在黑色畫面和所有燈（包括觸控欄和鍵盤）關閉時即為完全關閉。

  2. 在 Mac 上按住電源按鈕，直到顯示系統卷宗和「選項」按鈕。

  3. 按下電源按鈕三次。

    注意：每次按下電源按鈕後，下一次需於 1 秒內按下。如你太快按下電源按鈕，其將不會登錄，因快速按下已保留作其他用途（如「旁白」）。

  4. 使用「藍牙設定輔助程式」來將藍牙配件與 Mac 配對。

    注意：如你已連接輸入裝置，將不會看到「藍牙設定輔助程式」。

  5. 選擇管理員帳户，然後按一下「下一步」。

  6. 輸入管理員帳户的密碼，然後按一下「繼續」。

    當「還原」App 在選單列上顯示，你可以選擇在視窗或選單列上任何可用的選項。

在安全模式開機

  1. 在 Mac 上，選擇「蘋果」選單 >「關機」。

    等待 Mac 完全關閉。Mac 在黑色畫面和所有燈（包括觸控欄和鍵盤）關閉時即為完全關閉。

  2. 在 Mac 上按住電源按鈕，直到顯示系統卷宗和「選項」按鈕。

  3. 選擇一個卷宗。

  4. 按住 Shift 鍵，然後在「安全模式」中按一下「繼續」。

    電腦會自動重新啟動。

設定預設開機卷宗並重新啟動電腦

  1. 在 Mac 上，選擇「蘋果」選單 >「關機」。

    等待 Mac 完全關閉。Mac 在黑色畫面和所有燈（包括觸控欄和鍵盤）關閉時即為完全關閉。

  2. 在 Mac 上按住電源按鈕，直到顯示系統卷宗和「選項」按鈕。

  3. 選擇一個卷宗。

  4. 按住 Option 鍵，然後按一下「總是使用」。

    電腦會自動重新啟動。

使用 Wi-Fi 連接到互聯網

如螢幕右上角的 Wi-Fi 選單圖像包含一或多條列 ，Wi-Fi 即為開啟並已連接到無線網絡。如 Wi-Fi 選單圖像並不包含任何列 ，Wi-Fi 已開啟但未連接到無線網絡。如有斜線穿過 Wi-Fi 選單圖像上 ，即 Wi-Fi 已關閉。Wi-Fi 需要開啟才能加入無線網絡。

在你的 Mac 上，執行以下任何操作：

  • 開啟 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「開啟 Wi-Fi」。

  • 關閉 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「關閉 Wi-Fi」。

  • 加入網絡：按一下 Wi-Fi 選單圖像，然後選擇網絡。如系統要求，請輸入網絡的密碼。

  • 加入封閉式網絡：按一下 Wi-Fi 選單圖像，選後選擇「加入其他網絡」。輸入網絡的名稱，並在系統要求時，輸入網絡密碼。

修復內置儲存空間

如你的電腦發生問題或如你開機後看到「macOS 還原」視窗，你可能需要修復電腦的內置儲存空間。

  1. 在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。

  2. 在「磁碟工具程式」App 中，選擇「顯示方式」>「顯示所有裝置」。

  3. 在側邊欄中，選擇卷宗，然後按一下「修理工具」按鈕 。

  4. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  5. 在「修理工具」程序完成後，按一下「完成」。

  6. 為儲存裝置上的每個卷宗重覆步驟 3 至 5。

  7. 在側邊欄中，選擇容器，然後按一下「修理工具」按鈕 。

  8. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  9. 在「修理工具」程序完成後，按一下「完成」。

  10. 為儲存裝置上的每個容器重覆步驟 7 至 9。

  11. 在側邊欄中，選擇儲存裝置，然後按一下「修理工具」按鈕 。

  12. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  13. 在「修理工具」程序完成後，按一下「完成」。

如「磁碟工具程式」無法修復你的磁碟，請盡量備份資料，然後遵循清除和重新安裝 macOS 的指示執行。

如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。

使用「時光機」來還原檔案

如你之前已製作「時光機」備份，你可以從備份還原檔案。

重要事項：只能使用「時光機」備份將資料還原至原來備份的來源電腦上。如你要從這部 Mac 傳送資料到新的電腦上，請結束「macOS 還原」，重新啟動 Mac，然後使用 macOS 的「系統轉移輔助程式」。

  1. 如果備份位於「時間囊」，請確定已連接互聯網。

  2. 在「還原」App 中，選擇「從時光機還原」，然後按一下「繼續」。

  3. 在「時光機系統還原」App 中，遵循螢幕上的指示操作。

重新安裝 macOS

在部份情況下，你可能需要重新安裝 macOS。你可以重新安裝 macOS 同時完整保留你的檔案和用户設定。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，按一下「重新安裝 macOS [名稱]」，然後按一下「繼續」。

  3. 遵循螢幕上的指示操作。

    重要事項：當被要求選擇磁碟時，選擇你目前的 macOS 磁碟（在大部份情況下，這是唯一選項）。

清除並重新安裝 macOS

如你清除並重新安裝 macOS，Mac 上你的用户帳户、網絡設定、所有檔案和資料夾都將刪除。清除 macOS 前，請嘗試登入返回你的系統，然後將要保留的檔案複製到其他儲存裝置來備份。也請記下「網絡」設定中的網絡設定，以便重新安裝 macOS 後可以再次輕鬆連線。

注意：清除和重新安裝 macOS 時，可能會安裝之前版本的 macOS。如要從安裝的版本升級至最新版本的 macOS，請前往「蘋果」選單 >「系統設定」，在側邊欄中按一下「一般」（你可能需要向下捲動），然後在右邊按一下「軟件更新」。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，按一下「磁碟工具程式」，然後按一下「繼續」。

  3. 在「磁碟工具程式」中，於側邊欄選擇你要清除的卷宗，然後按一下工具列的「清除」按鈕。

  4. 在「名稱」欄位輸入卷宗名稱。

  5. 按一下「格式」彈出式選單，選擇 APFS，然後按一下「清除卷宗群組」。

  6. 清除程序完成後，按一下「完成」，然後選擇「磁碟工具程式」>「結束磁碟工具程式」。

  7. 在「還原」App 中，按一下「重新安裝 macOS [名稱]」，按一下「繼續」，然後遵循螢幕上的指示操作。

更改保安政策

  1. 在「還原」App 中，選擇「磁碟工具程式」> 「開機保安工具程式」。

  2. 選擇你要用於設定保安政策的系統。

  3. 如磁碟已加密，按一下「解鎖」，輸入密碼，然後按一下「解鎖」。

  4. 按一下「保安政策」。

  5. 選擇以下一個保安政策選項：

    • 完整保安：確保只有你目前的作業系統或 Apple 目前信任的已簽署作業系統軟件可以執行。此模式需要在軟件安裝時連接網絡。

    • 較低保安：允許執行 Apple 曾經信任的任何版本的已簽署作業系統軟件。

  6. 如你選擇「較低保安」，請按需要選擇以下任何選項：

    • 允許用户管理來自已識別開發者的核心延伸功能：允許安裝使用舊版核心延伸功能的軟件。

    • 允許遙距管理核心延伸功能和自動軟件更新：使用流動裝置管理（MDM）解決方案授權遙距管理舊版核心延伸功能和軟件更新。

  7. 按一下「好」。

  8. 如你更改保安政策，按一下「用户」彈出式選單，選擇管理員帳户，輸入管理員帳户密碼，然後按一下「好」。

  9. 選擇「蘋果」選單 >「重新開機」。

    你必須重新啟動 Mac 才能讓更改生效。

使用「分享磁碟」以在兩部 Mac 電腦之間傳送檔案

透過共用卷宗並使用 USB、USB-C 或 Thunderbolt 連接線連接兩部電腦，你可以在啟動到「macOS 還原」的 Mac 和另一部 Mac 之間傳送檔案。

  1. 使用 USB、USB-C 或 Thunderbolt 連接線連接兩部電腦。

  2. 在「還原」App 中，選擇「磁碟工具程式」>「分享磁碟」。

  3. 選擇你要分享的卷宗，然後按一下「開始分享」。

  4. 在另一部 Mac 上，開啟 Finder 視窗，然後在側邊欄中按一下「位置」下方的「網絡」。

  5. 在「網絡」視窗中，按兩下已分享磁碟的 Mac，按一下「連接身份」，在「連接身份」視窗中選擇「訪客」，然後按一下「連線」。

  6. 傳送檔案。

  7. 當你完成傳送檔案後，退出另一部 Mac 上的卷宗。

  8. 在「分享磁碟」App 中，按一下「停止分享」。

  9. 選擇「分享磁碟」>「結束分享磁碟」。

檢視還原記錄

還原記錄包含由「macOS 還原」記錄的訊息。你可以指定要顯示的訊息類別、過濾訊息以搜尋特定文字，並將還原記錄儲存至檔案。

  1. 在「還原」App 中，選擇「視窗」>「還原記錄」。

  2. 按一下「詳細層級」彈出式選單，然後選擇要顯示的訊息：

    • 只顯示錯誤：選擇此選項以只顯示錯誤訊息。

    • 顯示錯誤與過程：選擇此選項以顯示錯誤訊息和過程訊息。

    • 顯示所有記錄：選擇此選項以顯示所有訊息。

  3. 如要過濾訊息，在「過濾」欄位輸入你要搜尋的文字。

  4. 如要儲存還原記錄，按一下「儲存」按鈕，為檔案選擇位置，然後按一下「儲存」。

  5. 完成檢視還原記錄後，按一下視窗左上角的「關閉」按鈕 。

製作還原診斷檔案

如你使用「用户意見助理」提交問題，系統可能會要求你提供還原診斷檔案。你可以製作檔案並將其儲存到外置儲存裝置。

  1. 將外置儲存裝置連接到 Mac。

  2. 按下 Control＋Option＋Shift＋Command＋句號鍵。

  3. 在畫面閃爍後，稍等片刻，直到對話框出現。

  4. 使用對話框來將還原診斷檔案儲存到外置儲存裝置的位置中。

結束「macOS 還原」

從「時光機」還原系統或重新安裝 macOS 時，你的 Mac 會自動重新啟動。在其他情況下，你需要手動結束「macOS 還原」（例如，如果要在重新安裝 macOS 之前備份檔案，或只是使用「開機保安工具程式」變更了設定）。

在 Mac 上，執行以下一項操作：

  • 重新啟動 Mac：選擇「蘋果」選單 >「重新開機」。

  • 重新啟動 Mac 前，選擇一個不同的開機磁碟：選擇「蘋果」選單 >「開機磁碟」，選擇開機磁碟，然後按一下「重新開機」。

  • 關閉 Mac：選擇「蘋果」選單 >「關機」。
