Appleシリコンを搭載したMacでmacOS復旧を使用する

macOS復旧は、Macに内蔵されている復旧用システムです。

Appleシリコンを搭載したMacでmacOS復旧でアプリを使って、内蔵ストレージを修復する、macOSを再インストールする、Time Machineバックアップからファイルを復元する、別のストレージデバイスにセキュリティポリシーを設定する、2台のMacコンピュータ間でファイルを転送する、セーフモードで起動するといったことなどができます。

macOSの再インストールにはインターネット接続が必要です。インターネットに接続するには、ワイヤレスネットワークまたは有線ネットワークによる接続を利用できます。喫茶店、図書館、ホテルなどのキャプティブポータルまたは企業用ネットワークを介してWi-Fiに接続しようとしている場合は、macOS復旧ではインターネットにアクセスできないことがあります。“Wi-Fiを使ってインターネットに接続する”を参照してください。

注記: お使いのMacの下取り、売却、譲渡の準備をする場合は、macOSの消去アシスタントをご利用ください。消去アシスタントを使用するには、macOSでコンピュータを起動し、アップルメニュー＞“システム設定”と選択して、サイドバーで“一般”をクリック（下へのスクロールが必要な場合があります）、右側の“転送またはリセット”をクリック、それから“すべてのコンテンツと設定を消去”ボタンをクリックして、画面の指示に従ってください。

Appleシリコンを搭載したMacのmacOS復旧で利用可能なアプリ

Appleシリコンを搭載したMacのmacOS復旧では、以下のアプリを利用できます:

  • 復旧: 復旧アプリにより、macOS復旧のその他のアプリにアクセスできます。“macOS復旧でコンピュータを起動する”、“macOS復旧でコンピュータを起動してBluetooth入力デバイスをペアリングする”、および“macOS復旧を終了する”を参照してください。

  • Time Machine System Restore: Time Machineバックアップからデータを復元します。復旧アプリで、“Time Machineから復元”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“Time Machine System Restore”＞“Time Machine System Restoreを終了”と選択します。“Time Machineを使ってファイルを復元する”を参照してください。

  • macOSインストール: コンピュータにmacOSを再インストールします。復旧アプリで、“macOS [name]を再インストール”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“macOS [name]インストール”＞“macOSインストールを終了”と選択します。“macOSを再インストールする”を参照してください。

  • Safari: 復旧アプリで、“Safari”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“Safari”＞“Safariを終了”と選択します。インターネットに接続している場合は、Safariを使ってAppleサポートWebサイトで詳しい情報を検索できます。

  • ディスクユーティリティ: 内蔵ストレージを修復または消去します。復旧アプリで、“ディスクユーティリティ”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。“macOSを消去して再インストールする”および“内蔵ストレージを修復する”を参照してください。

  • 起動セキュリティユーティリティ: Macのセキュリティポリシーを設定します。復旧アプリで、“ユーティリティ”＞“起動セキュリティユーティリティ”と選択します。復旧アプリに戻るには、“起動セキュリティユーティリティ”＞“起動セキュリティユーティリティを終了”と選択します。“セキュリティポリシーを変更する”を参照してください。

  • ターミナル: コマンドラインで設定を変更します。復旧アプリで、“ユーティリティ”＞“ターミナル”と選択します。復旧アプリに戻るには、“ターミナル”＞“ターミナルを終了”と選択します。

  • ディスク共有: macOS復旧で起動したMacのディスクを共有します。復旧アプリで、“ユーティリティ”＞“ディスク共有”と選択します。“ディスク共有を使って2台のMacコンピュータ間でファイルを転送する”を参照してください。

  • 起動ディスク: Macの起動ディスクを設定します。アップルメニュー ＞“起動ディスク”と選択します。アプリを終了するには、“起動ディスク”＞“起動ディスクを終了”と選択します。

macOS復旧を使って、以下の作業を実施することもできます:

  • セーフモードでコンピュータを起動する

  • デフォルトの起動ボリュームを設定してコンピュータを再起動する

  • 復旧ログを表示する

  • 修復診断ファイルを作成する

macOS復旧でコンピュータを起動する

  1. Macで、アップルメニュー＞“システム終了”と選択します。

    Macが完全にシステム終了するまで待ちます。画面が黒くなり、Touch Barとキーボードを含むすべての明かりが消えると、Macのシステムが完全に終了したことになります。

  2. システムボリュームと“オプション”ボタンが表示されるまでMacの電源ボタンを押したままにします。

  3. “オプション”ボタンをクリックし、“続ける”をクリックします。

  4. もし尋ねられたら、復旧するボリュームを選択して“次へ”をクリックします。

  5. 管理者アカウントを選択して、“次へ”をクリックします。

  6. 管理者のパスワードを入力して、“次へ”をクリックします。

    復旧アプリがメニューバーに表示されたら、ウインドウまたはメニューバーで使用可能なオプションを選ぶことができます。

macOS復旧でコンピュータを起動してBluetooth入力デバイスをペアリングする

macOS復旧での起動時にBluetooth® キーボード、トラックパッド、またはマウスをMacとペアリングする必要がある場合は、以下の手順に従ってください。

  1. Macで、アップルメニュー＞ “システム終了”と選択します。

    Macが完全にシステム終了するまで待ちます。画面が黒くなり、Touch Barとキーボードを含むすべての明かりが消えると、Macのシステムが完全に終了したことになります。

  2. システムボリュームと“オプション”ボタンが表示されるまでMacの電源ボタンを押したままにします。

  3. 電源ボタンを3回押します。

    注記: 電源ボタンは1秒以内の間隔で押す必要があります。電源ボタンを押す速度が速すぎると、認識されません。高速に押す動作は、VoiceOverなど、ほかの目的用に確保されているためです。

  4. “Bluetooth設定アシスタント”を使用して、BluetoothアクセサリをMacとペアリングします。

    注記: すでに入力デバイスが接続されている場合は、“Bluetooth設定アシスタント”は表示されません。

  5. 管理者アカウントを選択して、“次へ”をクリックします。

  6. 管理者のパスワードを入力して、“次へ”をクリックします。

    復旧アプリがメニューバーに表示されたら、ウインドウまたはメニューバーで使用可能なオプションを選ぶことができます。

セーフモードでコンピュータを起動する

  1. Macで、アップルメニュー＞“システム終了”と選択します。

    Macが完全にシステム終了するまで待ちます。画面が黒くなり、Touch Barとキーボードを含むすべての明かりが消えると、Macのシステムが完全に終了したことになります。

  2. システムボリュームと“オプション”ボタンが表示されるまでMacの電源ボタンを押したままにします。

  3. ボリュームを選択します。

  4. Shiftキーを押したままにして、“セーフモードで続ける”をクリックします。

    コンピュータが自動的に再起動します。

デフォルトの起動ボリュームを設定してコンピュータを再起動する

  1. Macで、アップルメニュー＞“システム終了”と選択します。

    Macが完全にシステム終了するまで待ちます。画面が黒くなり、Touch Barとキーボードを含むすべての明かりが消えると、Macのシステムが完全に終了したことになります。

  2. システムボリュームと“オプション”ボタンが表示されるまでMacの電源ボタンを押したままにします。

  3. ボリュームを選択します。

  4. Optionキーを押したままにして、“常に使用”をクリックします。

    コンピュータが自動的に再起動します。

Wi-Fiを使ってインターネットに接続する

Wi-Fiメニューアイコン（画面の右上に表示される）に1本以上のバーが表示されている場合、Wi-Fiはオンになっていて、ワイヤレスネットワークに接続しています。Wi-Fiメニューアイコンにバーが表示されていない場合、Wi-Fiはオンになっていますが、ワイヤレスネットワークに接続していません。Wi-Fiメニューアイコンに斜線が入っている場合、Wi-Fiはオフになっています。ワイヤレスネットワークに接続する前に、Wi-Fiをオンにする必要があります。

Macで、以下のいずれかを実行します:

  • Wi-Fiをオンにする: Wi-Fiメニューアイコンをクリックしてから、“Wi-Fiをオンにする”を選択します。

  • Wi-Fiをオフにする: Wi-Fiメニューアイコンをクリックしてから、“Wi-Fiをオフにする”を選択します。

  • ネットワークに接続する: Wi-Fiメニューアイコンをクリックしてから、ネットワークを選択します。ネットワークのパスワードを求められたら、入力します。

  • 非公開ネットワークに接続する: Wi-Fiメニューアイコンをクリックしてから、“ほかのネットワークに接続”を選択します。ネットワークの名前と、求められた場合はネットワークのパスワードを入力します。

内蔵ストレージを修復する

コンピュータに問題がある場合、またはコンピュータの起動時に“macOS復旧”ウインドウが表示された場合は、コンピューの内蔵ストレージを修復する必要があると考えられます。

  1. 復旧アプリで、“ディスクユーティリティ”を選択してから、“続ける”をクリックします。

  2. ディスクユーティリティアプリで、“表示”＞“すべてのデバイスを表示”と選択します。

  3. サイドバーでボリュームを選択してから、“First Aid”ボタン をクリックします。

  4. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します。

  5. First Aidプロセスが完了したら、“完了”をクリックします。

  6. ストレージデバイス上のボリュームごとに、手順3〜5を繰り返します。

  7. サイドバーでコンテナを選択してから、“First Aid”ボタンをクリックします。

  8. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します

  9. First Aidプロセスが完了したら、“完了”をクリックします。

  10. ストレージデバイス上のコンテナごとに、手順7〜9を繰り返します。

  11. サイドバーでストレージデバイスを選択してから、“First Aid”ボタンをクリックします。

  12. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します。

  13. First Aidプロセスが完了したら、“完了”をクリックします。

ディスクユーティリティでディスクを修復できない場合は、できるだけ多くの情報のバックアップを作成してから、“macOSを消去して再インストールする”の手順に従ってください。

復旧アプリに戻るには、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します

Time Machineを使ってファイルを復元する

Time Machineバックアップを作成済みの場合は、バックアップからファイルを復元できます。

重要: バックアップ作成元のコンピュータに情報を復元する場合にのみ、Time Machineバックアップを使用してください。情報をこのMacから新しいコンピュータに転送する場合は、macOS復旧を終了し、Macを再起動してから、macOSの移行アシスタントアプリを使用します。

  1. バックアップがTime Capsuleにある場合は、インターネットに接続していることを確認します。

  2. 復旧アプリで、“Time Machineから復元”を選択してから、“続ける”をクリックします。

  3. Time Machine System Restoreアプリで、画面に表示される指示に従います。

macOSを再インストールする

状況によっては、macOSの再インストールが必要な場合があります。ファイルとユーザ設定をそのままにしてmacOSを再インストールできます。

  1. インターネットに接続していることを確認します。

  2. 復旧アプリで、“macOS [name]を再インストール”をクリックしてから、“続ける”をクリックします。

  3. 画面に表示される指示に従います。

    重要: ディスクを選択するように求められたとき、現在のmacOSディスクを選択します（ほとんどの場合は、利用可能なディスクは1つだけです）。

macOSを消去して再インストールする

macOSを消去して再インストールする場合は、ユーザアカウント、ネットワーク設定、およびMac上のすべてのファイルとフォルダが削除されます。macOSを消去する前に、システムへの再ログインを試み、残しておきたいファイルを別のストレージデバイスにコピーすることでそれらのバックアップを作成してください。また、“ネットワーク”設定からネットワーク設定を書き留めておくと、macOSの再インストール後の再接続が簡単になります。

注記: macOSを消去して再インストールすると、以前のバージョンのmacOSがインストールされることがあります。インストールされているバージョンを最新バージョンのmacOSにアップグレードするには、アップルメニュー >システム設定に移動し、サイドバーにある“一般”をクリック（下へのスクロールが必要な場合があります）, それから右側の“ソフトウェアアップデート”をクリックします。

  1. インターネットに接続していることを確認します。

  2. 復旧アプリで、“ディスクユーティリティ”をクリックしてから、“続ける”をクリックします。

  3. “ディスクユーティリティ”のサイドバーで消去したいボリュームを選択して、ツールバーの“消去”ボタンをクリックします。

  4. “名前”フィールドにボリュームの名前を入力します。

  5. “フォーマット”ポップアップメニューをクリックし、“APFS”を選択して、“ボリュームグループを消去”をクリックします。

  6. 消去プロセスが完了したら、“完了”をクリックし、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。

  7. 復旧アプリで、“macOS [name]を再インストール”を選択し、“続ける”をクリックしてから、画面に表示される指示に従います。

セキュリティポリシーを変更する

  1. 復旧アプリで、“ユーティリティ”＞“起動セキュリティユーティリティ”と選択します。

  2. セキュリティポリシーの設定に使用したいシステムを選択します。

  3. ディスクが暗号化されている場合は、“ロックを解除”をクリックし、パスワードを入力してから、“ロックを解除”をクリックします。

  4. “セキュリティポリシー”をクリックします。

  5. 以下のセキュリティオプションを確認します:

    • 完全なセキュリティ: 現在のOS、または現在Appleによって信頼されている署名付きオペレーティング・システム・ソフトウェアのみを実行できるようにします。このモードでは、ソフトウェアのインストール時にネットワーク接続が必要です。

    • 低セキュリティ: これまでにAppleによって信頼された署名付きオペレーティング・システム・ソフトウェアのすべてのバージョンに対して実行を許可します。

  6. “低セキュリティ”を選択した場合、必要に応じて、以下のいずれかを実行します:

    • 確認済みの開発元から提供されたカーネル機能拡張のユーザ管理を許可: 従来のカーネル拡張を使用するソフトウェアのインストールを許可します。

    • カーネル機能拡張と自動ソフトウェア・アップデートのリモート管理を許可: モバイルデバイス管理（MDM）ソリューションを使用して、レガシーカーネル拡張およびソフトウェア・アップデートのリモート管理を許可します。

  7. “OK”をクリックします。

  8. セキュリティポリシーを変更した場合は、“ユーザ”ポップアップメニューをクリックし、管理者アカウントを選択し、管理者アカウントのパスワードを入力して“OK”をクリックします。

  9. アップルメニュー ＞“再起動”と選択します。

    変更が有効になるように、Macを再起動する必要があります。

ディスク共有を使って2台のMacコンピュータ間でファイルを転送する

ボリュームを共有し、2台のコンピュータをUSB、USB-C、またはThunderboltケーブルで接続することで、macOS復旧で起動したMacと別のMacの間でファイルを転送できます。

  1. 2台のコンピュータをUSB、USB-C、またはThunderboltケーブルで接続します。

  2. 復旧アプリで、“ユーティリティ”＞“ディスク共有”と選択します。

  3. 共有したいボリュームを選択してから、“共有を開始”をクリックします。

  4. もう1台のMacで、Finderウインドウを開いてから、サイドバーで“場所”の下にある“ネットワーク”をクリックします。

  5. “ネットワーク”ウインドウで、共有ディスクがあるMacをダブルクリックし、“別名で接続”をクリックして、“別名で接続”ウインドウで“ゲスト”を選択してから、“接続”をクリックします。

  6. ファイルを転送します。

  7. ファイルの転送が完了したら、もう1台のMacでボリュームを取り出します。

  8. ディスク共有アプリで、“共有を停止”をクリックします。

  9. “ディスク共有”＞“ディスク共有を終了”と選択します。

復旧ログを表示する

復旧ログには、macOS復旧によってログに記録されたメッセージが含まれています。表示するメッセージの種類を指定したり、メッセージにフィルタを適用して特定のテキストを検索したり、復旧ログをファイルに保存したりできます

  1. 復旧アプリで、“ウインドウ”＞“復旧ログ”と選択します。

  2. “詳細レベル”ポップアップメニューをクリックしてから、表示するメッセージを選択します:

    • エラーのみ表示: エラーメッセージだけを表示するには、このオプションを選択します。

    • エラーと進行状況を表示: エラーメッセージと進行状況メッセージを表示するには、このオプションを選択します。

    • すべてのログを表示: すべてのメッセージを表示するには、このオプションを選択します。

  3. メッセージにフィルタを適用するには、“フィルタ”フィールドに検索したいテキストを入力します。

  4. 復旧ログを保存するには、“保存”ボタンをクリックし、ファイルの場所を選択してから、“保存”をクリックします。

  5. 復旧ログの確認が終わったら、ウインドウの左上隅にある“閉じる”ボタンをクリックします。

修復診断ファイルを作成する

フィードバックアシスタントを使って問題レポートを登録する場合に、修復診断ファイルの提出を求められることがあります。このファイルを作成して、外部ストレージデバイスに保存できます。

  1. 外部ストレージデバイスをMacに接続します

  2. Control＋Option＋Shift＋Command＋ピリオド（.）キーを押します。

  3. スクリーンが点滅したら、ダイアログが表示されるまで、しばらく待ちます。

  4. ダイアログを使用して、修復診断ファイルを外部ストレージデバイス上の場所に保存します。

macOS復旧を終了する

Time Machineからのシステムの復元またはmacOSの再インストールを行うと、その間にMacが自動的に再起動します。それ以外の場合は、手動でmacOS復旧を終了する必要があります（例えば、macOSを再インストールする前にファイルのバックアップを作成したい場合や、起動セキュリティユーティリティを使って設定を変更した場合など）。

Macで、以下のいずれかを実行します:

  • Macを再起動する: アップルメニュー＞“再起動”と選択します。

  • 別の起動ディスクを選択してからMacを再起動する: アップルメニュー＞“起動ディスク”と選択し、起動ディスクを選択してから、“再起動”をクリックします。

  • Macをシステム終了する: アップルメニュー＞“システム終了”と選択します。
