Informations pour les utilisateurs invités sur Mac

Lorsque vous ouvrez une session en tant qu’invité sur un Mac sur lequel FileVault est activé, vous ne pouvez utiliser que Safari. Les autres apps installées sur le Mac ne sont pas accessibles.

Lorsque vous fermez la session, tous les fichiers que vous avez créés sont supprimés.

Se connecter à Internet

Si l’icône de menu Wi-Fi (dans le coin supérieur droit de l’écran) contient une ou plusieurs barres , cela signifie que le Wi-Fi est activé et connecté à un réseau sans fil. Si l’icône de menu Wi-Fi ne contient aucune barre , cela signifie que le Wi-Fi est activé mais n’est pas connecté à un réseau sans fil. Si l’icône de menu Wi-Fi est barrée , cela signifie que le Wi-Fi est désactivé. Le Wi-Fi doit être activé avant que vous puissiez vous connecter à un réseau sans fil.

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Pour activer ou désactiver le Wi-Fi : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Activer le Wi-Fi » ou « Désactiver le Wi-Fi ».

  • Pour vous connecter à un réseau : cliquez sur l’icône de menu Wi-Fi, puis choisissez un réseau. Si vous y êtes invité, saisissez le mot de passe du réseau.

Mettre fin à une session « Utilisateur invité »

Une fois que vous avez terminé d’utiliser le Mac, effectuez l’une des opérations suivantes :

  • Pour redémarrer l’ordinateur : choisissez le menu Pomme  > Redémarrer, ou choisissez Safari > Quitter Safari.

  • Pour éteindre l’ordinateur : choisissez le menu Pomme  > Éteindre.
