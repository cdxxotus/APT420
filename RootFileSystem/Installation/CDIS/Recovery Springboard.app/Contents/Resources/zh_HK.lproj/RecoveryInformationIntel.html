在 Intel 基準的 Mac 上使用 macOS 還原

「macOS 還原」是 Mac 的內置還原系統。

你可以使用「macOS 還原」中的 App 來在 Intel 基準的 Mac 上修復內置儲存裝置、重新安裝 macOS、從「時光機」備份還原檔案、設定保安選項等等。

重新安裝 macOS 需要互聯網連線。如要連接互聯網，你可以使用無線或有線網絡連線。如你正在嘗試透過限制入口網站（例如咖啡店、圖書館或酒店）或企業網絡連接 Wi-Fi，你可能無法在「macOS 還原」中取用互聯網。請參閲：使用 Wi-Fi 連接互聯網。

注意：如你想讓 Mac 準備好以舊換新、出售或贈送，並且你的電腦是以 Intel 為基準配備 Apple T2 安全晶片的 Mac，請在 macOS 中使用「清除輔助程式」。如要使用「清除輔助程式」，請在 macOS 中啟動電腦，選擇「蘋果」選單 >「系統設定」，在側邊欄中按一下「一般」（你可能需要向下捲動），按一下右邊的「轉移或重設」，按一下「清除所有內容和設定」按鈕，然後按照螢幕上的指示操作。

在 Intel 基準的 Mac 上「macOS 還原」中可用的 App

以下的 App 可在 Intel 基準的 Mac 上之「macOS 還原」中使用：

  • 還原：「還原」App 在你於「macOS 還原」中開機時會開啟。請參閲：在「macOS 還原」中開啟 Intel 基準的 Mac 和結束「macOS 還原」。

  • 時光機系統還原：從「時光機」備份還原資料。在「還原」App 中，選擇「從時光機還原」，然後按一下「繼續」。如要返回「還原」App， 選擇「時光機系統還原」>「結束時光機系統還原」。請參閲：使用「時光機」來還原你的檔案。

  • 安裝 macOS：在你的電腦上重新安裝 macOS。 在「還原」App 中，選擇「重新安裝 macOS [名稱]」，然後按一下「繼續」。如要返回「還原」App， 選擇「安裝 macOS [名稱]」>「結束安裝 macOS」。請參閲：重新安裝 macOS。

  • Safari：在「還原」App 中，選擇 Safari，然後按一下「繼續」。如要返回「還原」App，選擇 Safari >「結束 Safari」。如你已連接到互聯網，你可以使用 Safari 來在 Apple 支援網站上搜尋更多資料。

  • 磁碟工具程式：修復或清除磁碟。在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。請參閲：清除和重新安裝 macOS。

  • 開機保安工具程式：為你的 Mac 設定保安政策。在「還原」App 中，選擇「工具程式」>「開機保安工具程式」。如要返回「還原」App， 選擇「開機保安工具程式」>「結束開機保安工具程式」。請參閲：使用「開機保安工具程式」來設定韌體密碼和使用「開機保安工具程式」來設定安全開機選項。

    根據你的 Mac 型號，「開機保安工具程式」中可用的選項會有所不同。請參閲 Apple 支援文章：關於配備 Apple T2 保安晶片的 Mac 上的「開機保安工具程式」和配備 Apple T2 安全晶片的 Mac 電腦。

  • 開機磁碟：為 Mac 設定開機磁碟。選擇「蘋果」選單 >「開機磁碟」。如要返回「還原」App， 選擇「開機磁碟」>「結束開機磁碟」。

  • 終端機：在「還原」App 中，選擇「工具程式」>「終端機」。如要返回「還原」App， 選擇「終端機」>「結束終端機」。

你也可以使用「macOS 還原」來執行以下操作：

  • 檢視還原記錄

  • 製作還原診斷檔案

在 Intel 基準的 Mac 之「macOS 還原」中開機

  1. 重新啟動 Mac。

  2. 立即按住以下組合鍵之一，直到看到啟動畫面：

    • Command + R：從內置的「macOS 還原」系統啟動。 使用此組合鍵可以重新安裝系統上已安裝的最新 macOS，或使用「macOS 還原」中的其他 App。

    • Option + Command + R：透過互聯網從「macOS 還原」啟動。使用此組合鍵可以重新安裝 macOS 並升級到與 Mac 相容的最新版本 macOS。

    • Option + Shift + Command + R：透過互聯網從「macOS 還原」啟動。使用此組合鍵重新安裝 Mac 隨附的 macOS 版本或 Mac 仍適用的最接近版本。

  3. 如你看到鎖頭，請輸入 Mac 的密碼。

  4. 如果磁碟上有多個卷宗，請選擇要還原的卷宗，然後按一下「下一步」。

  5. 如系統要求，請選擇一個管理員帳户，按一下「下一步」，輸入該帳户的密碼，然後按一下「繼續」。

    當「還原」App 在選單列上顯示，你可以選擇在視窗或選單列上任何可用的選項。

使用 Wi-Fi 連接到互聯網

如螢幕右上角的 Wi-Fi 選單圖像包含一或多條列 ，Wi-Fi 即為開啟並已連接到無線網絡。如 Wi-Fi 選單圖像並不包含任何列 ，Wi-Fi 已開啟但未連接到無線網絡。如有斜線穿過 Wi-Fi 選單圖像上 ，即 Wi-Fi 已關閉。Wi-Fi 需要開啟才能加入無線網絡。

在你的 Mac 上，執行以下任何操作：

  • 開啟 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「開啟 Wi-Fi」。

  • 關閉 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「關閉 Wi-Fi」。

  • 加入網絡：按一下 Wi-Fi 選單圖像，然後選擇網絡。如系統要求，請輸入網絡的密碼。

  • 加入封閉式網絡：按一下 Wi-Fi 選單圖像，選後選擇「加入其他網絡」。輸入網絡的名稱，並在系統要求時，輸入網絡密碼。

修復內置儲存裝置

如你的電腦發生問題或如你開機後顯示「macOS 還原」視窗，你可能需要修復電腦的內置儲存裝置 (例如固態硬碟 (SSD)）。

  1. 在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。

  2. 在「磁碟工具程式」App 中，選擇「顯示方式」>「顯示所有裝置」。

  3. 在側邊欄中，選擇卷宗，然後按一下「修理工具」按鈕 。

  4. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  5. 在「修理工具」程序完成後，按一下「完成」。

  6. 為儲存裝置上的每個卷宗重覆步驟 3 至 5。

  7. 在側邊欄中，選擇容器，然後按一下「修理工具」按鈕 。

  8. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  9. 在「修理工具」程序完成後，按一下「完成」。

  10. 為儲存裝置上的每個容器重覆步驟 7 至 9。

  11. 在側邊欄中，選擇儲存裝置，然後按一下「修理工具」按鈕 。

  12. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  13. 在「修理工具」程序完成後，按一下「完成」。

如「磁碟工具程式」無法修復你的磁碟，請盡量備份資料，然後遵循清除和重新安裝 macOS 的指示執行。

如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。

使用「時光機」來還原檔案

如你之前已製作「時光機」備份，你可以從備份還原檔案。

重要事項：只能使用「時光機」備份將資料還原至原來備份的來源電腦上。如你要從這部 Mac 傳送資料到新的電腦上，請結束「macOS 還原」，重新啟動 Mac，然後使用 macOS 的「系統轉移輔助程式」。

  1. 如果備份位於「時間囊」，請確定已連接互聯網。

  2. 在「還原」App 中，選擇「從時光機還原」，然後按一下「繼續」。

  3. 在「時光機系統還原」App 中，遵循螢幕上的指示操作。

使用「開機保安工具程式」來設定韌體密碼

設定韌體密碼會阻止你的 Mac 從你選擇的開機磁碟以外的任何內置或外置儲存裝置開機（除非輸入韌體密碼）。你可以使用「開機保安工具程式」開啟、關閉或更改韌體密碼。

開啟韌體密碼後，每次從「macOS 還原」或不是從「開機磁碟」設定中選擇的儲存裝置啟動時，都需要輸入韌體密碼。

重新啟動 Mac 之後，對韌體密碼的更改才會生效。

  1. 在「還原」App 中，選擇「工具程式」>「開機保安工具程式」。

  2. 如系統要求，請輸入管理員帳户和密碼。

  3. 執行以下任何操作：

    • 開啟韌體密碼：按一下「開啟韌體密碼」，輸入新密碼，重新輸入密碼來驗證密碼，然後按一下「設定密碼」。

    • 關閉韌體密碼：按一下「關閉韌體密碼」，輸入密碼，然後按一下「關閉密碼」。

    • 更改韌體密碼：按一下「更改密碼」，輸入舊密碼，輸入新密碼並進行驗證，然後按一下「更改密碼」。

  4. 按一下「結束開機保安工具程式」。

使用「開機保安工具程式」來設定安全開機選項

安全開機選項只在具有 Apple T2 安全晶片的 Mac 電腦上可用。

重新啟動 Mac 之後，對安全開機選項的更改將會生效。

  1. 在「還原」App 中，選擇「磁碟工具程式」>「開機保安工具程式」。

  2. 輸入管理員用户名稱和密碼。

  3. 在「安全開機」下方選擇一個保安選項。

    • 完整保安：確保只有你目前的 OS 或 Apple 目前信任的已簽署作業系統軟件可以執行。此模式需要在軟件安裝時連接網絡。

    • 中等保安：允許執行 Apple 曾經信任的任何版本的已簽署作業系統軟件。

    • 無保安：在可開機作業系統上不強制執行任何要求。

  4. 在「允許開機媒體」下方選擇一個開機選項。

    • 不允許從外置或可移除媒體開機：限制從不受 Apple T2 安全晶片保護的任何裝置（如 USB 和 Thunderbolt 磁碟機或內部已連接的 PCIe 或 SATA 磁碟機）開機的能力。

    • 允許從外置或可移除媒體開機：不限制從任何裝置開機的能力。

  5. 選擇「開機保安工具程式」>「結束開機保安工具程式」。

重新安裝 macOS

在部份情況下，你可能需要重新安裝 macOS。你可以重新安裝 macOS 同時完整保留你的檔案和用户設定。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，按一下「重新安裝 macOS [名稱]」，然後按一下「繼續」。

  3. 遵循螢幕上的指示操作。

    重要事項：當被要求選擇磁碟時，選擇你目前的 macOS 磁碟（在大部份情況下，這是唯一選項）。

清除並重新安裝 macOS

如你清除並重新安裝 macOS，Mac 上你的用户帳户、網絡設定，以及所有檔案和資料夾都將刪除。清除 macOS 前請先依照下方指示，嘗試登入返回你的系統，然後將要保留的任何檔案複製到其他儲存裝置來備份。也請記下「網絡」設定中的網絡設定，以便重新安裝 macOS 後可以再次輕鬆連線。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，按一下「磁碟工具程式」，然後按一下「繼續」。

  3. 在「磁碟工具程式」中，於側邊欄選擇你要清除的卷宗，然後按一下工具列的「清除」按鈕。

  4. 在「名稱」欄位輸入卷宗名稱。

  5. 按一下「格式」彈出式選單，選擇 APFS，然後按一下「清除卷宗群組」。

  6. 清除程序完成後，按一下「完成」，然後選擇「磁碟工具程式」>「結束磁碟工具程式」。

  7. 在「還原」App 中，按一下「重新安裝 macOS [名稱]」，按一下「繼續」，然後遵循螢幕上的指示操作。

檢視還原記錄

還原記錄包含由「macOS 還原」記錄的訊息。你可以指定要顯示的訊息類別、過濾訊息以搜尋特定文字，並將還原記錄儲存至檔案。

  1. 在「還原」App 中，選擇「視窗」>「還原記錄」。

  2. 按一下「詳細層級」彈出式選單，然後選擇要顯示的訊息：

    • 只顯示錯誤：選擇此選項以只顯示錯誤訊息。

    • 顯示錯誤與過程：選擇此選項以顯示錯誤訊息和過程訊息。

    • 顯示所有記錄：選擇此選項以顯示所有訊息。

  3. 如要過濾訊息，在「過濾」欄位輸入你要搜尋的文字。

  4. 如要儲存還原記錄，按一下「儲存」按鈕，為檔案選擇位置，然後按一下「儲存」。

  5. 完成檢視還原記錄後，按一下視窗左上角的「關閉」按鈕 。

製作還原診斷檔案

如你使用 Feedback Assistant 提交問題報告，系統可能會要求你提供還原診斷檔案。你可以製作檔案並將其儲存到外置儲存裝置。

  1. 將外置儲存裝置連接到 Mac。

  2. 按下 Control + Option + Shift + Command + 句號鍵。

  3. 在畫面閃爍後，稍等片刻，直到對話框出現。

  4. 使用對話框來將還原診斷檔案儲存到外置儲存裝置的位置中。

結束「macOS 還原」

從「時光機」還原系統或重新安裝 macOS 時，你的 Mac 會自動重新啟動。在其他情況下，你需要手動結束「macOS 還原」（例如，如果要在重新安裝 macOS 之前備份檔案，或只是使用「開機保安工具程式」變更了設定）。

在 Mac 上，執行以下一項操作：

  • 重新啟動 Mac：選擇「蘋果」選單 >「重新開機」。

  • 重新啟動 Mac 前，選擇一個不同的開機磁碟：選擇「蘋果」選單 >「開機磁碟」，選擇開機磁碟，然後按一下「重新開機」。

  • 關閉 Mac：選擇「蘋果」選單 >「關機」。
