Intelプロセッサ搭載のMacでmacOS復旧を使用する

macOS復旧は、Macに内蔵されている復旧用システムです

Intelプロセッサが搭載されたMacのmacOS復旧でアプリを使って、内蔵ストレージデバイスを修復する、macOSを再インストールする、Time Machineバックアップからファイルを復元する、セキュリティオプションを設定するといったことなどができます。

macOSの再インストールにはインターネット接続が必要です。インターネットに接続するには、ワイヤレスネットワークまたは有線ネットワークによる接続を利用できます。喫茶店、図書館、ホテルなどのキャプティブポータルまたは企業用ネットワークを介してWi-Fiに接続しようとしている場合は、macOS復旧ではインターネットにアクセスできないことがあります。“Wi-Fiを使ってインターネットに接続する”を参照してください。

注記: お使いのMacがApple T2セキュリティチップ搭載のIntelプロセッサが搭載されたMacで、下取り、売却、譲渡の準備をする場合は、macOSの消去アシスタントをご利用ください。消去アシスタントを使用するには、macOSでコンピュータを起動し、アップルメニュー＞“システム設定”と選択して、サイドバーで“一般”をクリック（下へのスクロールが必要な場合があります）、右側の“転送またはリセット”をクリック、それから“すべてのコンテンツと設定を消去”ボタンをクリックして、画面の指示に従ってください。

Intelプロセッサ搭載のMacのmacOS復旧で利用可能なアプリ

Intelプロセッサ搭載のMacのmacOS復旧では、以下のアプリを利用できます:

  • 復旧: macOS復旧でコンピュータが起動すると、復旧アプリが開きます。“macOS復旧でIntelプロセッサ搭載のMacを起動する”および“macOS復旧を終了する”を参照してください。

  • Time Machine System Restore: Time Machineバックアップからデータを復元します。復旧アプリで、“Time Machineから復元”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“Time Machine System Restore”＞“Time Machine System Restoreを終了”と選択します。“Time Machineを使ってファイルを復元する”を参照してください。

  • macOSインストール: コンピュータにmacOSを再インストールします。復旧アプリで、“macOS [name]を再インストール”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“macOS [name]インストール”＞“macOSインストールを終了”と選択します。“macOSを再インストールする”を参照してください。

  • Safari: 復旧アプリで、“Safari”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“Safari”＞“Safariを終了”と選択します。インターネットに接続している場合は、Safariを使ってAppleサポートWebサイトで詳しい情報を検索できます。

  • ディスクユーティリティ: ディスクを修復または消去します。復旧アプリで、“ディスクユーティリティ”を選択してから、“続ける”をクリックします。復旧アプリに戻るには、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。“macOSを消去して再インストールする”を参照してください。

  • 起動セキュリティユーティリティ: Macのセキュリティポリシーを設定します。復旧アプリで、“ユーティリティ”＞“起動セキュリティユーティリティ”と選択します。復旧アプリに戻るには、“起動セキュリティユーティリティ”＞“起動セキュリティユーティリティを終了”と選択します。“起動セキュリティユーティリティを使ってファームウェアパスワードを設定する”および“起動セキュリティユーティリティを使って安全な起動のオプションを設定する”を参照してください。

    起動セキュリティユーティリティで利用可能なオプションは、お使いのMacの機種によって異なります。Appleのサポート記事“Apple T2 セキュリティチップを搭載したMacの起動セキュリティユーティリティについて”および“Apple T2セキュリティチップ搭載モデルのMac”を参照してください。

  • 起動ディスク: Macの起動ディスクを設定します。アップルメニュー＞“起動ディスク”と選択します。復旧アプリに戻るには、“起動ディスク”＞“起動ディスクを終了”と選択します。

  • ターミナル: 復旧アプリで、“ユーティリティ”＞“ターミナル”と選択します。復旧アプリに戻るには、“ターミナル”＞“ターミナルを終了”と選択します。

macOS復旧を使って以下の操作を行うこともできます:

  • 復旧ログを表示する

  • 修復診断ファイルを作成する

macOS復旧でIntelプロセッサ搭載のMacを起動する

  1. Macを再起動します。

  2. すぐに以下のいずれかのキーコンビネーションを押したままにして、起動画面が表示されるまで待ちます:

    • Command-Rキー: 内蔵のmacOS復旧システムから起動します。お使いのシステムにインストールされた最新のmacOSを再インストールするか、macOS復旧でほかのアプリを使用するときは、このキーコンビネーションを使用します。

    • Option-Command-Rキー: インターネット経由でmacOS復旧から起動します。macOSを再インストールして、お使いのMacと互換性のある最新バージョンのmacOSにアップグレードするときは、このキーコンビネーションを使用します。

    • Option-Shift-Command-Rキー: インターネット経由でmacOS復旧から起動します。お使いのMacの出荷時のバージョンのmacOS、またはそれに最も近い現在入手可能なバージョンを再インストールするときは、このキーコンビネーションを使用します

  3. カギが表示されたら、Macのパスワードを入力します。

  4. ディスクに複数のボリュームがある場合は、復旧したいボリュームを選択してから、“次へ”をクリックします。

  5. 求められた場合は管理者のアカウントを選択して、“次へ”をクリックし、アカウントのパスワードを入力してから、“続ける”をクリックします。

    復旧アプリがメニューバーに表示されたら、ウインドウまたはメニューバーで使用可能なオプションを選ぶことができます。

Wi-Fiを使ってインターネットに接続する

Wi-Fiメニューアイコン（画面の右上に表示される）に1本以上のバーが表示されている場合、Wi-Fiはオンになっていて、ワイヤレスネットワークに接続しています。Wi-Fiメニューアイコンにバーが表示されていない 、場合Wi-Fiはオンになっていますが、ワイヤレスネットワークに接続していません。Wi-Fiメニューアイコンに斜線が入っている場合、Wi-Fiはオフになっています。ワイヤレスネットワークに接続する前に、Wi-Fiをオンにする必要があります。

Macで、以下のいずれかを実行します:

  • Wi-Fiをオンにする: Wi-Fiメニューアイコンをクリックしてから、“Wi-Fiをオンにする”を選択します。

  • Wi-Fiをオフにする: Wi-Fiメニューアイコンをクリックしてから、“Wi-Fiをオフにする”を選択します。

  • ネットワークに接続する: Wi-Fiメニューアイコンをクリックしてから、ネットワークを選択します。ネットワークのパスワードを求められたら、入力します。

  • 非公開ネットワークに接続する: Wi-Fiメニューアイコンをクリックしてから、“ほかのネットワークに接続”を選択します。ネットワークの名前と、求められた場合はネットワークのパスワードを入力します。

内蔵ストレージデバイスを修復する

コンピュータに問題がある場合、またはコンピュータの起動時に“macOS復旧”ウインドウが表示された場合は、コンピューの内蔵ストレージデバイス（ソリッドステートドライブ（SSD）など）を修復する必要があると考えられます。

  1. 復旧アプリで、“ディスクユーティリティ”を選択してから、“続ける”をクリックします。

  2. ディスクユーティリティアプリで、“表示”＞“すべてのデバイスを表示”と選択します。

  3. サイドバーでボリュームを選択してから、“First Aid”ボタンをクリックします。

  4. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します

  5. First Aidプロセスが完了したら、“完了”をクリックします。

  6. ストレージデバイス上のボリュームごとに、手順3〜5を繰り返します。

  7. サイドバーでコンテナを選択してから、“First Aid”ボタンをクリックします。

  8. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します。

  9. First Aidプロセスが完了したら、“完了”をクリックします。

  10. ストレージデバイス上のコンテナごとに、手順7〜9を繰り返します。

  11. サイドバーでストレージデバイスを選択してから、“First Aid”ボタンをクリックします。

  12. “First Aid”ダイアログで、“実行”をクリックして、画面に表示される指示に従って操作します。

  13. First Aidプロセスが完了したら、“完了”をクリックします。

ディスクユーティリティでディスクを修復できない場合は、できるだけ多くの情報のバックアップを作成してから、“macOSを消去して再インストールする”の手順に従ってください。

復旧アプリに戻るには、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。

Time Machineを使ってファイルを復元する

Time Machineバックアップを作成済みの場合は、バックアップからファイルを復元できます。

重要: バックアップ作成元のコンピュータに情報を復元する場合にのみ、Time Machineバックアップを使用してください。情報をこのMacから新しいコンピュータに転送する場合は、macOS復旧を終了し、Macを再起動してから、macOSの移行アシスタントアプリを使用します。

  1. バックアップがTime Capsuleにある場合は、インターネットに接続していることを確認します。

  2. 復旧アプリで、“Time Machineから復元”を選択してから、“続ける”をクリックします。

  3. Time Machine System Restoreアプリで、画面に表示される指示に従います。

起動セキュリティユーティリティを使ってファームウェアパスワードを設定する

ファームウェアパスワードを設定すると、ファームウェアパスワードを入力しない限り、選択した起動ディスク以外の内蔵/外付けのストレージデバイスからはMacを起動できなくなります。起動セキュリティユーティリティを使って、ファームウェアパスワードをオンまたはオフにしたり、変更したりできます。

ファームウェアパスワードがオンになっていると、macOS復旧から、または“起動ディスク”設定で選択したもの以外のストレージデバイスから起動するたびに、ファームウェアパスワードを入力する必要があります。

Macを再起動すると、ファームウェアパスワードに対する変更が有効になります。

  1. 復旧アプリで、“ユーティリティ”＞“起動セキュリティユーティリティ”と選択します。

  2. 管理者のユーザ名とパスワードを求められたら、入力します。

  3. 以下のいずれかを実行します:

    • ファームウェアパスワードをオンにする: “ファームウェアパスワードをオンにする”をクリックし、パスワードを入力して、確認のためにパスワードを再入力してから、“パスワードを設定”をクリックします。

    • ファームウェアパスワードをオフにする: “ファームウェアパスワードをオフにする”をクリックし、パスワードを入力してから、“パスワードをオフにする”をクリックします。

    • ファームウェアパスワードを変更する: “パスワードを変更”をクリックし、現在のパスワードを入力し、新しいパスワードを入力して、確認してから、“パスワードを変更”をクリックします。

  4. “起動セキュリティユーティリティを終了”をクリックします。

起動セキュリティユーティリティを使って安全な起動のオプションを設定する

安全な起動のオプションは、Apple T2セキュリティチップを搭載したMacコンピュータでのみ使用できます。

Macを再起動すると、安全な起動のオプションに対する変更が有効になります。

  1. 復旧アプリで、“ユーティリティ”＞“起動セキュリティユーティリティ”と選択します。

  2. 管理者のユーザ名とパスワードを入力します。

  3. “安全な起動”の下のセキュリティオプションを選択します。

    • 完全なセキュリティ: 現在のOS、または現在Appleによって信頼されている署名付きオペレーティング・システム・ソフトウェアのみを実行できるようにします。このモードでは、ソフトウェアのインストール時にネットワーク接続が必要です。

    • 中程度のセキュリティ: これまでにAppleによって信頼された署名付きオペレーティング・システム・ソフトウェアのすべてのバージョンに対して実行を許可します。

    • セキュリティなし: 起動可能なオペレーティングシステムに要件を適用しません。

  4. “許可する起動メディア”の下の起動オプションを選択します。

    • 外部メディアまたはリムーバブルメディアからの起動を許可しない: T2で保護されていないデバイス（USBまたはThunderboltドライブや、内部で接続されているPCIeまたはSATAドライブなど）から起動できないように制限します。

    • 外部メディアまたはリムーバブルメディアからの起動を許可: どのデバイスからの起動も制限しません

  5. “起動セキュリティユーティリティ”＞“起動セキュリティユーティリティを終了”と選択します。

macOSを再インストールする

状況によっては、macOSの再インストールが必要な場合があります。ファイルとユーザ設定をそのままにしてmacOSを再インストールできます。

  1. インターネットに接続していることを確認します。

  2. 復旧アプリで、“macOS [name]を再インストール”をクリックしてから、“続ける”をクリックします。

  3. 画面に表示される指示に従います。

    重要: ディスクを選択するように求められた場合は、現在のmacOSディスクを選択します（ほとんどの場合は、利用可能なディスクは1つだけです）。

macOSを消去して再インストールする

macOSを消去して再インストールする場合は、ユーザアカウント、ネットワーク設定、およびMac上のすべてのファイルとフォルダが削除されます。macOSを消去する前に、システムへの再ログインを試み、残しておきたいファイルを別のストレージデバイスにコピーすることでそれらのバックアップを作成してください。また、“ネットワーク”設定からネットワーク設定を書き留めておくと、macOSの再インストール後の再接続が簡単になります。

  1. インターネットに接続していることを確認します。

  2. 復旧アプリで、“ディスクユーティリティ”をクリックしてから、“続ける”をクリックします。

  3. “ディスクユーティリティ”のサイドバーで消去したいボリュームを選択して、ツールバーの“消去”ボタンをクリックします。

  4. “名前”フィールドにボリュームの名前を入力します。

  5. “フォーマット”ポップアップメニューをクリックし、“APFS”を選択して、“ボリュームグループを消去”をクリックします。

  6. 消去プロセスが完了したら、“完了”をクリックし、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。

  7. 復旧アプリで、“macOS [name]を再インストール”を選択し、“続ける”をクリックしてから、画面に表示される指示に従います。

復旧ログを表示する

復旧ログには、macOS復旧によってログに記録されたメッセージが含まれています。表示するメッセージの種類を指定したり、メッセージにフィルタを適用して特定のテキストを検索したり、復旧ログをファイルに保存したりできます。

  1. 復旧アプリで、“ウインドウ”＞“復旧ログ”と選択します。

  2. “詳細レベル”ポップアップメニューをクリックしてから、表示するメッセージを選択します:

    • エラーのみ表示: エラーメッセージだけを表示するには、このオプションを選択します。

    • エラーと進行状況を表示: エラーメッセージと進行状況メッセージを表示するには、このオプションを選択します。

    • すべてのログを表示: すべてのメッセージを表示するには、このオプションを選択します。

  3. メッセージにフィルタを適用するには、“フィルタ”フィールドに検索したいテキストを入力します。

  4. 復旧ログを保存するには、“保存”ボタンをクリックし、ファイルの場所を選択してから、“保存”をクリックします。

  5. 復旧ログの確認が終わったら、ウインドウの左上隅にある“閉じる”ボタンをクリックします。

修復診断ファイルを作成する

フィードバックアシスタントを使って問題レポートを登録する場合に、修復診断ファイルの提出を求められることがあります。このファイルを作成して、外部ストレージデバイスに保存できます。

  1. 外部ストレージデバイスをMacに接続します。

  2. Control＋Option＋Shift＋Command＋ピリオド（.）キーを押します。

  3. スクリーンが点滅したら、ダイアログが表示されるまで、しばらく待ちます。

  4. ダイアログを使用して、修復診断ファイルを外部ストレージデバイス上の場所に保存します。

macOS復旧を終了する

Time Machineからのシステムの復元またはmacOSの再インストールを行うと、その間にMacが自動的に再起動します。それ以外の場合は、手動でmacOS復旧を終了する必要があります（例えば、macOSを再インストールする前にファイルのバックアップを作成したい場合や、起動セキュリティユーティリティを使って設定を変更した場合など）。

Macで、以下のいずれかを実行します:

  • Macを再起動する: アップルメニュー＞“再起動”と選択します。

  • 別の起動ディスクを選択してからMacを再起動する: アップルメニュー＞“起動ディスク”と選択し、起動ディスクを選択してから、“再起動”をクリックします。

  • Macをシステム終了する: アップルメニュー＞“システム終了”と選択します。
