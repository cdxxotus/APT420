在 Intel 架構式 Mac 上使用「macOS 復原」

「macOS 復原」是 Mac 上的內建復原系統。

你可以在 Intel 架構式 Mac 上使用「macOS 復原」中的 App 來修復內置儲存裝置、重新安裝 macOS、從「時光機」備份回復檔案、設定安全性選項等等。

重新安裝 macOS 需要網際網路連線。若要連接網際網路，你可以使用無線或有線網路連線。若你嘗試透過網頁驗證入口（例如在咖啡廳、圖書館或飯店）或企業級網路來連接 Wi-Fi ，你可能無法在「macOS 復原」中連接網際網路。請參閱：使用 Wi-Fi 連接網際網路。

注意：若你想讓 Mac 準備好以舊換新、出售或贈送，並且你的電腦是配備 Apple T2 安全晶片的 Intel 基準 Mac，請使用 macOS 中的「清除輔助程式」。 若要使用「清除輔助程式」，請在 macOS 中啟動電腦，選擇「蘋果」選單 >「系統設定」，在側邊欄中按一下「一般」（可能需要往下捲動），按一下右側的「移轉或重置」，按一下「清除所有內容和設定」，然後依照螢幕上的指示操作。

在 Intel 架構式 Mac 上的「macOS 復原」中可用的 App

下列 App 可在 Intel 架構式 Mac 上的「macOS 復原」中使用：

  • 復原：當你的電腦在「macOS 復原」中啟動，「復原」App 便會打開。請參閱：在「macOS 復原」中啟動 Intel 架構式 Mac 和結束「macOS 復原」。

  • 時光機系統回復：從「時光機」備份回復你的資料。在「復原」App 中，選取「從時光機回復」，然後按一下「繼續」。若要返回「復原」App，請選擇「時光機系統回復」>「結束時光機系統回復」。請參閱：使用「時光機」來回復檔案。

  • 安裝 macOS ：在電腦上重新安裝 macOS。在「復原」App 中，選取「重新安裝 macOS [名稱]」，然後按一下「繼續」。若要返回「復原」App，請選擇「安裝 macOS [名稱]」>「結束安裝 macOS」。請參閱：重新安裝 macOS。

  • Safari：在「復原」App 中，選取 Safari，然後按一下「繼續」。若要返回「復原」App，請選擇 Safari >「結束 Safari」。若你已連接網際網路，便可使用 Safari 在 Apple 支援網站上搜尋更多資訊。

  • 磁碟工具程式：修復或清除磁碟。在「復原」App 中，選取「磁碟工具程式」，然後按一下「繼續」。若要返回「復原」App，請選擇「磁碟工具程式」>「結束磁碟工具程式」。請參閱：清除並重新安裝 macOS。

  • 開機安全性工具程式：設定 Mac 的安全性規則。在「復原」App 中，選擇「工具程式」>「開機安全性工具程式」。若要返回「復原」App，請選擇「開機安全性工具程式」>「結束開機安全性工具程式」。請參閱：使用「開機安全性工具程式」來設定韌體密碼和使用「開機安全性工具程式」來設定安全開機選項。

    「開機安全性工具程式」中的可用選項會視 Mac 機型而有所不同。請參閱 Apple 支援文章：「配備 Apple T2 安全晶片的 Mac 上的開機安全性工具程式」和「配備 Apple T2 安全晶片的 Mac 電腦」。

  • 啟動磁碟：設定 Mac 的啟動磁碟。選擇「蘋果」選單 >「啟動磁碟」。若要返回「復原」App，請選擇「啟動磁碟」>「結束啟動磁碟」。

  • 終端機：在「復原」App 中，選擇「工具程式」>「終端機」。若要返回「復原」App，請選擇「終端機」>「結束終端機」。

你也可以使用「macOS 復原」來執行以下作業：

  • 檢視復原記錄

  • 製作復原診斷檔案

在「macOS 復原」中啟動 Intel 架構式 Mac

  1. 重新啟動 Mac。

  2. 立即按住以下其中一個組合鍵直到你看見啟動畫面：

    • Command + R：從內建的「macOS 復原系統」啟動。使用此組合鍵來重新安裝系統已安裝的最新 macOS，或使用「macOS 復原」中的其他 App。

    • Option + Command + R：透過網際網路從「macOS 復原」啟動。使用此組合鍵來重新安裝 macOS 和升級到與你的 Mac 相容的最新版本 macOS。

    • Option + Shift + Command + R：透過網際網路從「macOS 復原」啟動。使用此組合鍵來重新安裝 Mac 隨附的 macOS 版本或仍適用的最接近版本。

  3. 若你看到一個鎖頭，請輸入 Mac 的密碼。

  4. 若你的磁碟上包含多個卷宗，請選取你要復原的卷宗，然後按「下一步」。

  5. 若系統要求，請選擇管理者帳號，按「下一步」，輸入帳號的密碼，然後按一下「繼續」。

    當「復原」App 顯示在選單列上時，你就可以選擇視窗或選單列上任何可用的選項。

使用 Wi-Fi 連接網際網路

若 Wi-Fi 選單圖像（在螢幕右上角）包含一格或多格訊號格數 ，表示 Wi-Fi 已開啟並連接無線網路。若 Wi-Fi 選單圖像未包含訊號格數 ，表示 Wi-Fi 已開啟但未連接無線網路。若 Wi-Fi 選單圖像有斜線穿過 ，表示 Wi-Fi 已關閉。需先開啟 Wi-Fi 才能加入無線網路。

在 Mac 上，執行以下任一操作：

  • 開啟 Wi-Fi： 按一下 Wi-Fi 選單圖像，然後選擇「開啟 Wi-Fi」。

  • 關閉 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「關閉 Wi-Fi」。

  • 加入網路：按一下 Wi-Fi 選單圖像，然後選擇一個網路。如有要求，請輸入網路的密碼。

  • 加入封閉網路：按一下 Wi-Fi 選單圖像，然後選擇「加入其他網路」。輸入網路的名稱和網路的密碼（如有要求）。

修復內部儲存裝置

若你的電腦發生問題或啟動電腦後顯示「macOS 復原」視窗，你可能需要修復電腦的內部儲存裝置，如固態硬碟（SSD）。

  1. 在「復原」App 中，選取「工具程式」，然後按一下「繼續」。

  2. 在「磁碟工具程式」App 中，選取「檢視」>「顯示所有裝置」。

  3. 在側邊欄中選取卷宗，然後按一下「修理工具」按鈕 。

  4. 在「修理工具」對話框中按一下「執行」，然後依照螢幕上的指示操作。

  5. 修理工具」程序執行完畢時，按一下「完成」。

  6. 為儲存裝置上的每一個卷宗重複執行步驟 3 到 5。

  7. 在側邊欄中選取容器，然後按一下「修理工具」按鈕 。

  8. 在「修理工具」對話框中按一下「執行」，然後依照螢幕上的指示操作。

  9. 「修理工具」程序執行完畢時，按一下「完成」。

  10. 為儲存裝置上的每一個容器重複執行步驟 7 到 9。

  11. 在側邊欄中選取儲存裝置，然後按一下「修理工具」按鈕 。

  12. 在「修理工具」對話框中按一下「執行」，然後依照螢幕上的指示操作。

  13. 「修理工具」程序執行完畢時，按一下「完成」。

若「磁碟工具程式」無法修復你的磁碟，請盡可能備份你的資訊，然後依照清除並重新安裝 macOS 中的指示操作。

若要返回「復原」App，請選擇「磁碟工具程式」>「結束磁碟工具程式」。

使用「時光機」來回復檔案

若你先前已製作「時光機」備份，就可以使用備份來回復檔案。

重要事項：請僅使用「時光機」備份將資料回復到原本備份的來源電腦上。若你要從這部 Mac 傳送資訊到新的電腦上，請結束「macOS 復原」，重新啟動 Mac，然後使用 macOS 中的「系統移轉輔助程式」App。

  1. 如果備份位於「時光囊」，請確定已連接網際網路。

  2. 在「復原」App 中，選取「從時光機回復」，然後按一下「繼續」。

  3. 在「時光機系統回復」App 中，依照螢幕上的指示操作。

使用「開機安全性工具程式」來設定韌體密碼

設定韌體密碼可防止 Mac 從任何你所選啟動磁碟以外的內置或外部儲存裝置啟動，除非輸入韌體密碼。你可以使用「開機安全性工具程式」開啟、關閉或更改韌體密碼。

若開啟了韌體密碼，每次當你從「macOS 復原」或「啟動磁碟」設定中所選磁碟以外的儲存裝置啟動時，都必須輸入韌體密碼。

重新啟動 Mac 後，韌體密碼的更動便會生效。

  1. 在「復原」App 中，選擇「工具程式」>「開機安全性工具程式」。

  2. 若系統要求，請輸入管理者的使用者名稱和密碼。

  3. 執行下列任一操作：

    • 開啟韌體密碼：按一下「開啟韌體密碼」，輸入新密碼，再次輸入新密碼來驗證，然後按一下「設定密碼」。

    • 關閉韌體密碼：按一下「關閉韌體密碼」，輸入密碼，然後按一下「關閉密碼」。

    • 更改韌體密碼：按一下「更改密碼」，輸入舊密碼，輸入新密碼並驗證，然後按一下「更改密碼」。

  4. 按一下「結束開機安全性工具程式」。

使用「開機安全性工具程式」來設定安全開機選項

安全開機選項只適用於配備 Apple T2 安全晶片的 Mac 電腦。

重新啟動 Mac 後，安全開機的更動便會生效。

  1. 在「復原」App 中，選擇「工具程式」>「開機安全性工具程式」。

  2. 輸入管理者的使用者名稱和密碼。

  3. 在「安全開機」下方選擇一個安全選項。

    • 完整安全性：確保只會執行目前的作業系統，或是 Apple 目前信任的已簽署作業系統軟體。此模式在軟體安裝期間需要網路連線。

    • 中等安全性：允許執行任何 Apple 所信任的已簽署作業系統軟體版本。

    • 無安全性：不會在可開機的作業系統上強制執行任何需求。

  4. 在「允許的開機媒體」下方選擇一個開機選項。

    • 不允許從外部或可卸除式媒體開機：限制從任何未受 T2 保護的裝置（如 USB 和 Thunderbolt 磁碟機）或 PCIe 或 SATA 內接磁碟開機的功能。

    • 允許從外部或可卸除式媒體開機：不限制從任何裝置上開機的能力。

  5. 選擇「開機安全性工具程式」>「結束開機安全性工具程式」。

重新安裝 macOS

在部分情況下，你可能需要重新安裝 macOS。重新安裝 macOS 時，你仍可以保留原有的檔案和使用者設定。

  1. 請確定已連接網際網路。

  2. 在「復原」視窗中，按一下「重新安裝 macOS [名稱]」，然後按一下「繼續」。

  3. 依照螢幕上的指示操作。

    重要事項： 當系統要求你選擇磁碟時，請選擇現用的 macOS 磁碟（在大部分情況下，這是唯一可選的磁碟）。

清除並重新安裝 macOS

若你清除並重新安裝 macOS，你電腦上的使用者帳號、網路設定、所有檔案和檔案夾都將刪除。清除 macOS 前，請先嘗試重新登入你的系統，將想要保留的檔案拷貝到其他儲存裝置來進行備份。也請記下「網路」設定中的網路設定，以便重新安裝 macOS 後可以再次輕鬆連線。

  1. 確定已連接。

  2. 在「復原」App 中，按一下「磁碟工具程式」，然後按一下「繼續」。

  3. 在「磁碟工具程式」的側邊欄中選取你要清除的卷宗，然後按一下工具列上的「清除」按鈕。

  4. 在「名稱」欄位中輸入卷宗的名稱。

  5. 按一下「格式」彈出式選單，選擇 APFS，然後按一下「清除卷宗群組」。

  6. 清除程序完成後，請按一下「完成」，然後選擇「磁碟工具程式」>「結束磁碟工具程式」。

  7. 在「復原」App 中按一下「重新安裝 macOS [名稱]」，按一下「繼續」，然後依照螢幕上的指示操作。

檢視復原記錄

復原記錄包含「macOS 復原」所記錄的訊息。你可以指定要檢視的訊息類型、過濾訊息以搜尋特定文字，以及將復原記錄儲存成檔案。

  1. 在「復原」App 中選擇「視窗」>「復原記錄」。

  2. 按一下「詳細層級」彈出式選單，然後選擇要顯示的訊息：

    • 只顯示錯誤：選擇此選項只會顯示錯誤訊息。

    • 顯示錯誤與過程：選擇此選項會顯示錯誤訊息和過程訊息。

    • 顯示所有記錄：選擇此選項會顯示所有訊息。

  3. 若要過濾訊息，請在「過濾」欄位中輸入你要搜尋的文字。

  4. 若要儲存復原記錄，請按一下「儲存」按鈕，選擇檔案位置，然後按一下「儲存」。

  5. 當你檢視完畢復原記錄時，請按一下視窗左上角的「關閉」 按鈕。

製作復原診斷檔案

若你使用「使用者意見程式」來製作回報問題的檔案，則系統可能會要求你提供復原診斷檔案。你可以製作此檔案並儲存在外接儲存裝置上。

  1. 連接外部儲存裝置到你的 Mac 上。

  2. 按下 Control + Option + Shift + Command + 句點鍵。

  3. 螢幕閃爍後，等待片刻直到顯示對話框。

  4. 使用對話框來將復原診斷檔案儲存到外接儲存裝置的位置上。

結束「macOS 復原」

從「時光機」回復系統或重新安裝 macOS 時，你的 Mac 會自動重新開機。在其他情況下，你需要手動結束「macOS 復原」（例如，若你要在重新安裝 macOS 前備份檔案，或是若你僅使用「開機安全性工具程式」更改了設定）。

在 Mac 上，執行下列其中一項操作：

  • 重新啟動 Mac：選擇「蘋果」選單 >「重新開機⋯」。

  • 在重新啟動 Mac 前選擇其他啟動磁碟：選擇「蘋果」選單 > 「啟動磁碟」，選取一個啟動磁碟，然後按一下「重新開機」。

  • 將 Mac 關機：選擇「蘋果」選單 >「關機⋯」。
